## 摘要
铰接物体操作对服务机器人而言是一项独特的挑战。现有方法采用端到端策略学习、视觉运动规划以及大型语言/视觉语言模型（LLM/VLM），但常常忽视了铰接物体的多样性和末端执行器与把手之间交互的复杂性，导致泛化能力有限和破坏性碰撞。为解决这一问题，我们提出了GSAM，一种通用且安全的铰接物体操作机器人框架。具体而言，一个基于视觉的感知器生成运动学参数。考虑到感知器中预训练标记产生的原始估计可能偏离常识，我们提出了一种基于VLM的精调优化器，利用思维链（COT）常识推理来优化感知。为防止破坏性碰撞，我们设计了一个交互约束函数生成器，将铰接物体、交互姿态和避障知识整合到一个基础中。然后LLM将这些约束功能化，并应用于轨迹和姿态规划。一个运动学感知的操作规划器验证轨迹和姿态的可达性。在5个物体类别的50个铰链任务和50个随机初始化的末端执行器-把手配置上的实验表明，与最佳基线相比，GSAM将标准差降低了3.1%，并将操作成功率提高了36.0%，分别证明了GSAM在实际场景中卓越的对象泛化能力和交互安全性。

## Abstract
Articulated object manipulation is a unique challenge for service robots. Existing methods employ end-to-end policy learning, visionmotion planning, and large-language/visual-language model (LLM/VLM), but often overlook the diversity of articulated objects and the complexity of interactions between end-effector and handle, leading to limited generalization and destructive collisions. To address this, we propose GSAM, a generalizable and safe robotic framework for articulated object manipulation. Specifically, a vision-based perceiver generates the kinematic parameters. Considering that pre-trained markers in perceiver yield raw estimations that may deviate from commonsense, we present a f ine-tuned VLM-based refiner, using chain-of-thought (COT) commonsense reasoning to refine perception. To prevent destructive collisions, we design an interaction constraint function generator, integrating articulated object, interaction pose, and obstacle avoidance knowledge into a base. LLM then functionalize these constraints and apply them to trajectory and posture planning. A kinematic-aware manipulation planner verifies reachability for trajectory and posture. Experiments on 50 hinge tasks across 5 object categories and 50 randomly initialized end-effectorhandle configurations show that GSAM reduces standard deviation by 3.1% and improves manipulation success rate by 36.0% compared to the best baseline, respectively demonstrating the superior object generalization and interaction safety of GSAM in practical scenarios.

## 1. 核心问题与整体含义（研究动机与背景）
- **研究背景**：服务机器人需要具备铰接物体操作能力（如开门、开抽屉），但现实中的铰接物体种类繁多（不同铰链类型、尺寸、纹理），且末端执行器与把手之间的交互关系复杂。
- **核心问题**：现有方法主要分为三类——端到端策略学习（如模仿学习、强化学习）、视觉-运动规划（如3DOI+运动规划）、大语言/视觉语言模型（LLM/VLM）引导。但普遍存在两个关键缺陷：
  1. **泛化性差**：模型针对特定物体训练，难以泛化到未见物体；外部视觉模型未经微调，估计结果常偏离常识（如关键点偏移、类别歧义）。
  2. **安全性不足**：忽略末端与把手的交互几何结构，在狭窄空间中易发生破坏性碰撞；现有方法往往只关注运动可行性或抓取规则之一，未系统整合多类知识。
- **研究意义**：发展一种既能泛化到多种铰接物体、又能保证交互安全的通用操作策略，对服务机器人的实际部署具有重要价值。

## 2. 方法论：GSAM框架
### 2.1 核心思想
GSAM采用模块化架构，将感知、常识推理、约束生成和运动学规划解耦，通过VLM的常识推理和LLM驱动的约束函数生成，提升泛化性和安全性。

### 2.2 关键技术细节
框架包含四个核心模块（图2）：
- **(a) Kinematic Parameter Perceiver (KPP)**  
  - 输入：RGB-D图像 + 关键点查询（来自SAM分割）。  
  - 流程：MAE预训练ViT编码图像→Transformer解码器聚合查询特征→多个预测头输出原始运动学参数（交互位置、铰链类型、物体掩码）。  
  - 输出：原始运动学参数 `K`。

- **(b) Kinematic Parameter Refiner (KPR)**  
  - 动机：KPP的原始估计常因遮挡、边界模糊等产生偏差。  
  - 方法：使用LoRA微调的LLaVA-7B作为VLM，设计校正提示集 `π_corr`（含演示、原始参数、任务指令、COT提示）。  
  - 核心：通过Chain-of-Thought（COT）逐步推理——先识别物体类别，再推断把手指向区域，最后结合视觉证据与结构先验优化关键点。  
  - 输出：多个候选精化参数集 `K_refined`（按置信度排序），公式 `K_refined = M_ref(o, K, π_corr)`，采用自回归交叉熵训练。

- **(c) Constraint Function Generator (CFG)**  
  - 知识库构建：将知识分为三类——`B_int`（交互姿态知识，含把手指向对应的末端姿态角）、`B_obs`（避障知识，基于RRT*生成的无碰撞轨迹模式）、`B_art`（铰接物体知识，通过最小二乘拟合得到开门/开抽屉轨迹函数）。  
  - 约束生成：使用LoRA微调的Llama 3.1-8B-Instruct，输入精化参数、知识库和COT提示，输出可执行的Python约束函数 `C`，分阶段（调整姿态、接近、打开）生成。  
  - 公式：`C = M_cfg(K_refined, B, π_c)`，同样采用自回归交叉熵微调。

- **(d) Kinematic-aware Manipulation Planner (KMP)**  
  - 坐标变换：将图像坐标转换为移动操作臂坐标系。  
  - 轨迹生成：在约束 `C` 下进行最小snap优化，生成N段多项式轨迹。  
  - 逆运动学：利用OMPL求解关节配置，满足关节限位和无碰撞条件。  
  - 回滚策略：若交互失败，依次尝试KPR中剩余的精化参数候选，循环直至成功或候选用尽。

### 2.3 算法流程（文字说明）
1. 输入RGB-D观测，KPP提取原始运动学参数K。
2. KPR利用微调VLM+ COT推理，生成多个精化参数候选K_refined。
3. CFG从知识库B中检索相关信息，由微调LLM生成分段约束函数C。
4. KMP进行坐标变换、约束轨迹优化、逆运动学求解，并执行，失败时回滚。
