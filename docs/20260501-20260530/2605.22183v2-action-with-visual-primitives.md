## 摘要
视觉-语言-动作（VLA）模型已成为通用机器人操作的一种有前景的范式。当前架构中常见的设计是将语言指令和视觉观察映射到单次前向传播中的动作。虽然概念上简单，但这种表述将指令理解、空间场景理解和运动控制纠缠在单一学习目标中。因此，动作专家必须隐式地重新学习预训练VLM中已存在的认知和感知能力，这可能限制学习效率和泛化能力。我们提出AVP（基于视觉基元的动作），一种端到端架构，实现了这种以视觉基元为中心的接口：VLM推断下一阶段目标并生成视觉基元令牌，这些令牌条件化一个流匹配动作专家，其监督来自末端执行器运动学。在通用抓取和放置任务上的真实机器人实验表明，AVP相比pi_0.5将成功率提高了27.61%，并优于其他近期方法，在数据效率、空间组合泛化和对象级迁移方面具有一致的优势。

## Abstract
Vision-Language-Action (VLA) models have emerged as a promising paradigm for generalist robotic manipulation. A common design in current architectures maps language instructions and visual observations to actions in a single forward pass. While conceptually simple, this formulation entangles instruction comprehension, spatial scene understanding, and motor control within a single learning objective. As a result, the action expert must implicitly relearn cognitive and perceptual capabilities already present in the pretrained VLM, which can limit both learning efficiency and generalization. We introduce AVP (Action with Visual Primitives), an end-to-end architecture that implements this visual-primitive-centric interface: the VLM infers the next-stage target and emits visual-primitive tokens that condition a flow-matching action expert, with supervision derived from end-effector kinematics. Real-robot experiments on general pick-and-place tasks show that AVP improves the success rate by 27.61% over pi_0.5 and outperforms other recent methods, with consistent gains in data efficiency, spatial-compositional generalization, and object-level transfer.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有Vision-Language-Action（VLA）模型通常将语言指令、视觉观察到动作的映射耦合在单次前向传播中，导致动作专家（action expert）需要隐式重新学习预训练VLM已经具备的语义理解和空间感知能力。这种耦合限制了学习效率和泛化能力，尤其在场景变化或对象布局改变时表现脆弱。
- **研究动机**：希望明确划分VLM与动作专家之间的职责——VLM负责“做什么、在哪里”，动作专家专注“如何执行”。当前方法包括：规划中心方法（如π₀.5使用子任务描述）仍受限于纯语言接口、视觉预测方法（如π₀.7提供未来帧）依然要求动作专家重新发现关键信息、视觉提示方法（如Point-VLA）依赖外部模型，导致流水线延迟和错误累积。
- **整体含义**：提出一种端到端、以视觉基元（visual primitives）作为显式通信接口的VLA架构，在VLM和动作专家之间建立清晰的、空间接地（spatially grounded）的中间表示，从而解耦认知与动作控制，提升效率和泛化。

## 2. 论文提出的方法论

### 2.1 核心思想
- **视觉基元接口**：VLM内部输出离散化的视觉基元（点、边界框、掩码、记忆等），这些基元被投影到视觉令牌空间，作为条件输入给动作专家，替代传统隐式接口或外部视觉提示。
- **端到端训练**：所有组件（VLM、视觉基元解码器、流匹配动作专家）联合优化，不需要外部检测器或分割器。
- **动作中心监督**：视觉基元的标注直接从末端执行器运动学通过关键帧提取、3D位姿获取、图像平面投影得到，无需人工标注。

### 2.2 关键技术细节
1. **VLM**：预训练的视觉语言模型，接收多视图观测和语言指令，输出多模态上下文令牌。
2. **视觉基元解码器**：自回归预测下一阶段的离散化视觉基元 \( p_t = D_\psi(\text{VLM}(o_t, l)) \)。
3. **投影操作**：将预测的视觉基元投影到视觉令牌空间，得到 \( z^{vp}_t = \text{Proj}(p_t, o_t) \)，与原始多模态令牌融合形成增强表示 \( z^{aug}_t \)。
4. **动作专家**：基于流匹配（flow matching）的扩散策略，以增强表示和本体感觉状态为条件，输出动作序列 \( a_{t:t+h} \)。

### 2.3 公式与损失
- 视觉基元预测损失：\( L_{vp} = L_{CE}(p_t, y^{vp}_t) \)，其中 \( y^{vp}_t \) 是从末端执行器运动学生成的ground-truth基元。

- 总损失：\( L = L_{act} + \lambda L_{vp} \)。

### 2.4 视觉基元监督生成流程（附录B）

## 7. 优点

- **方法创新**：提出视觉基元作为VLM与动作专家之间的显式、端到端学习接口，避免了外部模型和流水线错误累积，同时保持空间接地性。
- **监督自动生成**：从末端执行器运动学直接提取基元标签，无需人工标注或外部感知模型，提高了可扩展性和物理契合度。
- **强泛化**：在空间组合和跨域场景下表现显著优于基线，说明解耦有效。
- **高效部署**：推理延迟低（0.27秒），适合实时机器人控制。
- **实验设计全面**：涵盖多任务、多基线、消融和泛化评估，比较公平。

## 8. 不足与局限
