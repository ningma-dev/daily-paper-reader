## 摘要
视觉-语言-动作（VLA）模型随着更强的主干网络、更广泛的预训练和更大的示范数据集而快速发展，但其动作头仍然高度同质化：大多数直接预测固定世界坐标系中的动作命令。我们提出MCF-Proto，一种轻量级动作头，它赋予VLA策略一个运动中心动作框架（MCF）和基于原型的动作参数化。在每一步，策略预测一个旋转$R_t \in SO(3)$，从一组原型中在变换后的局部坐标系中组成动作，并将其映射回世界坐标系以实现端到端训练，仅使用标准示范数据，无需辅助监督。这种简单设计催生了稳定的涌现结构。在没有显式方向标签的情况下，学习到的局部坐标系发展出稳定的几何结构，其轴与示范的末端执行器运动高度兼容。同时，学习到的表示中的动作变得更加紧凑，变化由更少的占主导方向捕获，并由共享原型更规则地组织。这些结构特性转化为改进的鲁棒性，尤其是在几何扰动下。我们的结果表明，为动作头添加轻量级几何和组合结构可以实质性地改善VLA策略组织和泛化机器人操作行为的方式。补充材料中提供了匿名代码仓库。

## Abstract
Vision-Language-Action (VLA) models have advanced rapidly with stronger backbones, broader pre-training, and larger demonstration datasets, yet their action heads remain largely homogeneous: most directly predict action commands in a fixed world coordinate frame. We propose \textbf{MCF-Proto}, a lightweight action head that equips VLA policies with a Motion-Centric Action Frame (MCF) and a prototype-based action parameterization. At each step, the policy predicts a rotation $R_t \in SO(3)$, composes actions in the transformed local frame from a set of prototypes, and maps them back to the world frame for end-to-end training, using only standard demonstrations without auxiliary supervision. This simple design induces stable emergent structure. Without explicit directional labels, the learned local frames develop a stable geometric structure whose axes are strongly compatible with demonstrated end-effector motion. Meanwhile, actions in the learned representation become substantially more compact, with variation captured by fewer dominant directions and more regularly organized by shared prototypes. These structural properties translate into improved robustness, especially under geometric perturbations. Our results suggest that adding lightweight geometric and compositional structure to the action head can materially improve how VLA policies organize and generalize robotic manipulation behavior. An anonymized code repository is provided in the supplementary material.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：当前 Vision-Language-Action (VLA) 模型在感知和语言理解方面快速进步，但动作头（action head）高度同质化——几乎所有方法都直接在固定世界坐标系中回归末端执行器命令。这种参数化方式忽略了操控动作的**局部几何特性**：许多操作行为（如抓握、插入、旋转）在局部框架下具有相似结构，但在世界坐标系中呈现截然不同的向量，导致策略需要隐式吸收这些几何变化，限制了泛化性和鲁棒性。
- **启发**：操控动作本质上是局部的、与物体或交互相关的。如果策略能够学习一个观测条件化的局部坐标系，并在该坐标系下用可重用原型组合动作，可能使动作表示更紧凑、更易共享和迁移。
- **整体含义**：本文提出**动作参数化本身是一个重要的设计选择**，通过为动作头添加轻量级的几何和组合结构（学习局部帧 + 原型字典），可以显著改善 VLA 策略的行为组织和泛化能力，而不改变主干网络或训练数据。

## 2. 论文提出的方法论

### 2.1 核心思想
MCF-Proto 是一个轻量级动作头替换方案，包含两个关键组件：
- **Motion-Centric Action Frame (MCF)**：每步由策略特征预测一个旋转 \(R_t \in SO(3)\)，定义局部动作帧；动作先在局部帧中生成，再变换回世界帧进行端到端训练。
- **原型动作参数化**：在局部帧内，平移和旋转动作由小规模共享原型字典的线性组合构成，每个原型是一个 \(3 \times d\) 矩阵，通过门控机制选择并叠加。

### 2.2 关键技术细节
1. **局部帧预测**：使用轻量 MLP 和连续 6D 旋转表示从特征 \(h_t\) 预测 \(R_t\)。局部帧中的动作 \(\Delta x_t^{\text{mcf}}, \Delta r_t^{\text{mcf}}\) 通过 \(\Delta x_t = R_t \Delta x_t^{\text{mcf}}\) 等映射回世界帧。
2. **原型组合**：
   - 平移原型字典 \(\mathcal{B}^{\text{trans}} = \{B_k^{\text{trans}}\}_{k=1}^{K_t}\)，旋转字典 \(\mathcal{B}^{\text{rot}} = \{B_k^{\text{rot}}\}_{k=1}^{K_r}\)，\(K_t = K_r = 16\)。
   - 从特征 \(h_t\) 预测缩放向量 \(z^{\text{trans}}, z^{\text{rot}} \in \mathbb{R}^d\)（\(d=3\)）和门控 logits，经 softmax 得到门控分布 \(\pi_t^x, \pi_t^r\)。
   - 局部动作：\(\Delta x_t^{\text{mcf}} = \sum_k \pi_t^x[k] B_k^{\text{trans}} z^{\text{trans}}\)，旋转类似。
3. **正则化**：
   - 正交正则化 \(L_{\text{ortho}}\)：防止原型退化，保持矩阵列正交。
   - 时序平滑正则化 \(L_{\text{smooth}}\)：鼓励相邻帧旋转 \(R_t\) 变化平滑（基于旋转矩阵迹的余弦损失）。
4. **损失函数**：最终动作监督为平移和夹爪 L1 损失 + 旋转 Smooth L1 损失 + 两项正则化。

### 2.3 算法流程（文字描述）
1. 观测 \(o_t\) 经 VLA 主干得特征 \(h_t\)。
2. 从 \(h_t\) 预测旋转 \(R_t\)（MCF）。
3. 从 \(h_t\) 预测缩放向量和门控分布，在局部帧内用原型组合得到 \(\Delta x_t^{\text{mcf}}, \Delta r_t^{\text{mcf}}\)。
4. 将局部动作变换回世界帧：\(\Delta x_t = R_t \Delta x_t^{\text{mcf}}\) 等。
5. 用世界帧动作目标 \(a_t^*\) 计算损失，端到端训练。

## 7. 优点

- **设计简洁有效**：仅改动动作头，不改变主干或训练范式，即取得显著提升，具有实用性和易集成性。
- **诱发涌现结构**：无需方向标签，学习到的帧自动对齐到运动主方向，动作表示在局部帧中自然紧凑，揭示出隐含的几何规律。
- **鲁棒性突出**：尤其对几何扰动（视角、初始位姿变化）增益大，符合设计预期——局部帧吸收场景变化后再解码动作。
- **诊断充分**：通过定量指标（协方差、有效秩、对齐角度）和可视化（轴使用、原型分布）深入解释机制，不止于性能数字。
- **开源代码**：提供匿名仓库，可重复性高。

## 8. 不足与局限

- **领域覆盖有限**：仅测试 LIBERO 和简单真实场景，未在高度动态、可变形物体、精细接触任务（如柔软物体、装配）上验证。LIBERO-plus 的扰动类别也不包含时间延迟、传感器故障等。
- **局部帧缺乏语义**：目前是隐式学习，未显式绑定到物体或接触点，在需要精确对准的任务中可能不如显式物体坐标系稳定。

- **旋转监督平滑性**：时序平滑损失依赖于连续旋转假设，在快速转向动作中可能过度约束，需要调节超参数。
