## 摘要
本文提出了RepWAM，一种以表征为中心的世界动作模型（WAM），它基于表征视觉-动作分词器构建。现有的WAM通常从预训练的视频生成模型中继承面向重建的视频分词器。虽然这些分词器保留了视觉保真度，但仅靠像素重建对于学习连接未来预测与机器人控制的指令遵循动态所提供的指导有限。为解决这一问题，我们探索了一种用于以表征为中心的世界动作建模的语义视觉-动作潜在空间。具体来说，我们训练了一个表征视觉-动作分词器，将视觉输入映射为对齐的视觉和潜在动作标记。然后，我们预训练WAM，使其在语言指令下联合建模未来视觉状态以及连接这些状态的潜在动作，随后适应真实机器人轨迹以实现闭环操作。在真实世界操作任务和仿真基准上的实验表明，RepWAM在多种操作场景中表现出强大的性能，而消融实验则凸显了语义视觉-动作分词化相比面向重建替代方案的价值。这些结果确立了表征视觉-动作分词化作为世界动作模型的一个有前景的基础，并朝向通用机器人策略迈出了一步。代码和权重将在https://github.com/wdrink/RepWAM上提供。

## Abstract
This work presents RepWAM, a representation-centric world action model (WAM) built on representation visual-action tokenizers. Existing WAMs typically inherit reconstruction-oriented video tokenizers from pretrained video generation models. Although these tokenizers preserve visual fidelity, pixel reconstruction alone provides limited guidance for learning instruction-following dynamics that connect future prediction with robot control. To address this, we explore a semantic visual-action latent space for representation-centric world action modeling. Specifically, we train a representation visual-action tokenizer that maps visual inputs into aligned visual and latent action tokens. We then pretrain our WAM to jointly model future visual states and the latent actions that connect them under language instructions, followed by adaptation to real robot trajectories for closed-loop manipulation. Experiments on real-world manipulation tasks and simulation benchmarks show that RepWAM delivers strong performance across diverse manipulation settings, while ablations highlight the value of semantic visual-action tokenization over reconstruction-oriented alternatives. These results establish representation visual-action tokenization as a promising foundation for world action models and a step toward generalist robot policies. Code and weights will be available at https://github.com/wdrink/RepWAM.

## 1. 论文的核心问题与整体含义（研究动机和背景）

现有世界动作模型（World Action Models, WAMs）通常从预训练的视频生成模型中继承面向重建的视频分词器（reconstruction-oriented video tokenizers）。这类分词器虽然保留了视觉保真度（像素重建质量），但存在两个关键缺陷：
- **视觉方面**：重建驱动的分词器将潜在容量浪费在低级外观（如背景纹理）上，而对操纵任务至关重要的物体身份、空间关系和交互线索表示不足，导致指令条件的世界模型在语义上难以对齐。
- **动作方面**：视觉潜在表示与机器人动作命令处于分离的空间，迫使逆动力学模型（IDM）每一步都需要跨越模态鸿沟。

因此，当前WAM的潜在空间在视觉上浅层且与动作结构上解耦。论文的核心问题是：**世界动作模型到底需要什么样的表征？** 答案是需要语义丰富的、与动作对齐的潜在空间。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
构建一个以表征为中心的世界动作模型（RepWAM），其基础是**表征视觉-动作分词器（RepViTok）**。该分词器将视觉潜在表示与冻结的视觉基础模型（Perception Encoder）对齐，获得语义丰富的视觉标记；同时学习一个潜在动作分词器（Latent Action Tokenizer, LAT），在同一个语义空间内通过耦合的逆动力学和正动力学模型诱导潜在动作，从而缩小视觉潜在与机器人动作之间的模态差距。

### 关键技术细节

#### (1) 表征视觉-动作分词器（RepViTok）
- **视觉分词器**：基于Vision Transformer (ViT) 的自动编码器。将第一帧分割为16×16 patches，后续帧分割为4×16×16 tubelets。编码器使用因果时间掩码和全空间注意力，解码器对称。损失函数：
  \[
  \mathcal{L}_{\text{rec}} = \lambda_1\|o-\hat{o}\|_1 + \lambda_{\text{perc}}\mathcal{L}_{\text{perc}}(o,\hat{o}) + \lambda_{\text{gan}}\mathcal{L}_{\text{gan}}(\hat{o})
  \]
  此外加入对齐损失：\(\mathcal{L}_{\text{align}} = \|W_{\text{align}} z - \text{avg}(G(o))\|_2^2\)，其中 \(G\) 是冻结的视觉基础模型（Perception Encoder）。总损失 \(\mathcal{L}_{\text{vis}} = \mathcal{L}_{\text{rec}} + \lambda_{\text{align}}\mathcal{L}_{\text{align}}\)。

- **潜在动作分词器**：冻结视觉分词器，训练一个耦合逆动力学模型（IDM）\(q_\phi\) 和正动力学模型（FDM）\(f_\psi\)。对连续潜在帧 \(z_t, z_{t+1}\)，IDM压缩过渡为潜在动作 \(\ell_t\)；FDM生成传输矩阵 \(K_t\) 和残差 \(\delta_t\)，预测下一潜在帧：
  \[
  \ell_t = q_\phi(z_t, z_{t+1}), \quad K_t, \delta_t = f_\psi(z_t, \ell_t), \quad \hat{z}_{t+1} = K_t z_t + \delta_t
  \]
  训练使用前向预测损失和反向一致性损失：
  \[
  \mathcal{L}_{\text{fwd}} = \sum_t \|\hat{z}_{t+1} - z_{t+1}\|_2^2, \quad \mathcal{L}_{\text{cons}} = \sum_t \|\hat{z}_t - z_t\|_2^2
  \]

#### (2) 因果世界动作模型
基于对齐的视觉和动作标记，训练因果扩散Transformer，进行联合视觉-动作生成，受语言指令条件。采用块因果掩码（block-causal mask），每个块内视觉和动作标记共享注意力权重但使用模态特定的前馈网络。训练损失为条件流匹配（Flow Matching）目标，同时优化视觉和动作分量：
\[
\mathcal{L}_{\text{FM}} = \mathbb{E}\left[ \|F_\theta^v(x_\alpha,\alpha,s_{<t}) - \dot{x}_\alpha^v\|_2^2 + \lambda_a \|F_\theta^a(x_\alpha,\alpha,s_{<t}) - \dot{x}_\alpha^a\|_2^2 \right]
\]

**总结流程**：RepViTok分词器预训练 → WAM预训练（联合建模未来视觉和潜在动作）→ 在真实机器人轨迹上微调适应（闭环操纵）。
