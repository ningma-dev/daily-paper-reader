## 摘要
视觉-语言-动作（VLA）策略将自然语言引入闭环机器人控制，使机器人能够直接根据文本指令执行操作任务。同一接口使文本在控制中具有重复作用，因为提示在每次重新规划步骤中被重复使用，而每个基于提示的动作都会改变策略所作用的未来观测。现有的VLA攻击研究了对抗性提示，这些提示能引发目标性的低层动作或使这些动作在变化的图像中持续存在。我们识别出一种更强的轨迹级故障模式：一个提示表面上仍指定了预期任务，但最终改变了物理结果。我们在数学上将这种设置形式化为“指令保持的轨迹重定向”，这是一种仅针对提示的威胁模型，其中攻击者在回合前选择一个提示，所有策略和环境组件保持固定，并且提示必须接近良性指令，同时省略目标词和纠正语言。为了找到这样的提示，我们引入了一种在线策略提示搜索方法，该方法使用rollouts来发现扰动，其闭环行为在满足指令保持约束的同时跟踪一个目标任务。在仿真和硬件上的实验表明，接近良性的提示扰动可以将VLA的rollouts重定向到攻击者指定的目标。这些结果揭示了VLA指令接地中的轨迹级脆弱性：表面上保持预期指令的文本仍然可以赋予对手对机器人最终物理结果的控制权。项目网站：https://vla-redirection-attack.github.io/

## Abstract
Vision-language-action (VLA) policies bring natural language into closed-loop robot control, enabling robots to execute manipulation tasks directly from text instructions. The same interface gives text a recurring role in control because the prompt is reused at every replanning step, and each prompt-conditioned action changes the future observations on which the policy acts. Existing VLA attacks study adversarial prompts that elicit targeted low-level actions or make such actions persist across changing images. We identify a stronger trajectory-level failure mode: a prompt that still $\textit{appears}$ to specify the intended task but redirects the final physical outcome. We mathematically formalize this setting as $\textit{command-preserving trajectory redirection}$, a prompt-only threat model in which the attacker chooses one prompt before the episode, all policy and environment components remain fixed, and the prompt must stay close to the benign instruction while omitting target words and correction language. To find such prompts, we introduce an on-policy prompt search method that uses rollouts to discover perturbations whose closed-loop behavior tracks a target task while satisfying the command-preserving constraints. Experiments in simulation and on hardware show that near-benign prompt perturbations can redirect VLA rollouts to attacker-specified targets. These results expose a trajectory-level vulnerability in VLA instruction grounding: text that appears to preserve the intended command can still give an adversary control over the robot's final physical outcome. ## 1. 核心问题与研究动机

- **研究背景**：视觉-语言-动作（VLA）策略将自然语言指令作为闭环机器人控制的持续输入，在每一步重新规划时重复使用。这种接口使非专家用户能通过语言操控机器人，但同时也引入了新的安全威胁。
- **核心问题**：现有攻击仅关注诱发特定低层动作或使动作在变化图像中持续存在，但无法控制机器人的最终物理结果。本文提出一种更强的轨迹级漏洞：一个表面上仍指定原任务的提示（即“指令保持”），能够重定向机器人的完整闭环轨迹，最终实现攻击者指定的替代物理目标。
- **整体含义**：VLA的指令接地过程存在轨迹级脆弱性——即使提示文本看似只做了微小修改且不含目标指令，攻击者仍可控制机器人的最终物理行为。这要求将语言接地视为闭环控制系统的组成部分进行安全防护。

## 2. 方法论

### 2.1 核心思想

- 定义“指令保持的轨迹重定向”威胁模型：攻击者在回合前选择一个提示，策略、环境、动力学均固定；该提示必须在文本上接近良性指令（如低编辑距离）、不含攻击目标词或纠正语言，但仍能诱导机器人执行攻击者指定的替代任务。
- 引入**在线策略提示搜索方法**：类似于 DAgger 的数据聚合思想，通过 rollout 收集候选提示所诱导的状态分布，然后用冻结的 VLA 策略对这些状态的观测重新标注良性/目标教师动作，并基于教师匹配损失优化提示。

### 2.2 关键技术细节

- **教师标签**：用同一冻结 VLA 在良性指令 τb 和目标指令 τt 下对观测 o 生成动作块 Ab(o) 和 At(o)。候选提示 τ 产生 Aτ(o)。

- **在线聚合**：初始数据集来自良性 rollout 和目标 rollout；每次迭代选取 top-M 候选进行 rollout，将其观测与教师标签加入数据集，更新评分分布。
- **约束检查**：通过文本成本、有效性、泄漏检查（禁止目标词/纠正语言）、保留检查（确保仍解读为原任务）过滤候选。

## 7. 优点

- **问题新颖**：首次形式化并展示 VLA 的轨迹级重定向威胁，比现有低层动作攻击更强，更贴近实际安全后果。
- **方法合理**：在线提示搜索借鉴模仿学习中的数据聚合思想，解决了固定观测评分不适用于闭环控制的问题；威胁模型定义清晰，约束严格（指令保持），避免 trivial 替换。
- **实验全面**：覆盖 9 种不同架构、仿真库和真实硬件，并提供因果分析、消融、防御评估，验证了漏洞的广泛性和机制。
- **可复现性**：提供了项目网站、代码、数据集和模型检查点，有助于后续研究。

## 8. 不足与局限

- **任务覆盖有限**：仅评估 LIBERO 桌面操作和 SO-100 简单操作，未涉及导航、移动操作、长周期多阶段任务，攻击表面可能不同。

- **缺少与其他攻击的对比**：未与 VLA-Fool、Jones et al. 等直接比较，难以量化新攻击的强度。
- **防御分析深度不足**：仅测试了简单预处理防御，未探索更鲁棒的鲁棒训练、对抗训练或认证防御；自适应攻击搜索可能进一步攻破某些防御。
