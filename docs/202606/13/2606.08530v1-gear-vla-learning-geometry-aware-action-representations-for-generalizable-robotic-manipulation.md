## 摘要
视觉-语言-动作（VLA）模型在基准测试中取得了强劲性能，但在实际部署中仍面临未见物体、背景变化和不同机器人本体的挑战。我们认为这源于缺乏统一的几何感知操作表征，使得现有VLA模型容易受到低级轨迹监督、未对齐的3D特征和本体差异的影响。为此，我们提出GEAR-VLA，一个用于学习统一几何感知动作表征以实现可泛化机器人操作的VLA框架。GEAR-VLA采用从粗到细的动作学习，其中多源具身预训练使视觉-语言模型（VLM）具备具身推理和离散动作理解能力，随后潜在动作标记将动作语义连接到梯度解耦的DiT连续动作专家模型。此外，通过将可训练的3D空间骨干网络与VLA表征对齐，同时冻结原始VLM对齐的视觉通路，实现语义对齐的3D集成。为了跨机器人共享该表征，GEAR-VLA使用本体规范化，将考虑本体的状态和本体不变的动作限制在低级接口中。广泛的仿真和真实世界实验展示了强大的泛化能力：GEAR-VLA在LIBERO、零样本LIBERO-Plus和RoboTwin 2.0上达到最先进性能，在AgileX上达到85.9%的成功率，在预训练未见过的LDT-01本体上达到81.0%，并在包含212个未见物体的6360次试验的通用抓取基准上达到90.1%的成功率。代码和模型将在https://github.com/babynabeauty/GEAR-VLA发布。

## Abstract
Vision-Language-Action (VLA) models achieve strong benchmark performance but still struggle in real-world deployment with unseen objects, background shifts, and different robot embodiments. We argue that this stems from the lack of a unified geometry-aware manipulation representation, leaving existing VLAs vulnerable to low-level trajectory supervision, misaligned 3D features, and embodiment differences. To address this, we propose GEAR-VLA, a VLA framework for learning unified geometry-aware action representations for generalizable robotic manipulation. GEAR-VLA adopts coarse-to-fine action learning, where multi-source embodied pretraining equips the VLM with embodied reasoning and discrete action understanding before latent action tokens connect action semantics to a gradient-decoupled DiT continuous action expert. It further performs semantic-aligned 3D integration by aligning a trainable 3D spatial backbone with the VLA representation while freezing the original VLM-aligned visual pathway. To share this representation across robots, GEAR-VLA uses embodiment canonicalization, where embodiment-aware states and embodiment-invariant actions confine robot differences to the low-level interface. Extensive simulation and real-world experiments demonstrate strong generalization: GEAR-VLA achieves state-of-the-art performance on LIBERO, zero-shot LIBERO-Plus, and RoboTwin 2.0, reaches 85.9% success on AgileX and 81.0% on the pretraining-unseen LDT-01 embodiment, and obtains 90.1% success on a 6,360-trial universal grasping benchmark with 212 unseen objects. Code and models will be released at https://github.com/babynabeauty/GEAR-VLA.

# GEAR-VLA 论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的视觉-语言-动作（VLA）模型在标准化基准上表现良好，但在实际部署中面对**未见物体、背景变化、不同机器人本体**时泛化能力严重不足。
- **根本原因**：现有模型缺乏**统一的几何感知动作表征**。具体表现为：
  - 动作表示方面：基于动作令牌（token）的VLA将连续控制量化成离散令牌，但容易退化为轨迹模仿而非具身推理。
  - 空间表示方面：现有方法引入的深度、3D位置编码或空间先验与VLM语义空间未对齐，直接注入会干扰语义表示。
  - 本体表示方面：使用机器人特定动作头或软提示（soft prompt）会将机器人身份嵌入共享策略表示，降低跨本体迁移效率，且对数据平衡要求高。
- **整体目标**：学习一个**语义基础且几何感知的统一动作表示**，同时将本体特定适配限制在低级接口中，从而实现跨场景、跨物体的可泛化机器人操作。

## 2. 方法论：核心思想、关键技术细节

### 2.1 总体架构
GEAR-VLA 接收多视角RGB图像、语言指令和机器人状态，输出未来动作块。核心由三大设计组成：**粗到细动作学习**、**语义对齐3D集成**、**本体规范化**。

### 2.2 粗到细动作学习

#### 阶段一：具身VLM预训练
- 先让VLM学习具身基础、任务理解和离散动作语义，**不直接接入连续动作**。
- 预训练数据混合：视觉-语言理解、空间基础、轨迹推理、指物、掩膜跟踪、操作视频等。
- 对**有动作标签的机器人数据**使用 **FAST 风格动作令牌**（将连续动作序列离散化为FAST令牌）提供离散监督。
- 对**无动作标签的操作视频**（如人类视频、网络视频）训练一个**因果VQ-VAE潜在动作分词器**：从连续视频帧中提取潜在动作ID，提供离散监督。这允许模型从无动作标签的视频中学到动作相关的视觉动态。
- 所有预训练任务统一为**自回归令牌预测**，损失函数为交叉熵：

\[
\mathcal{L}_{\text{VLM}} = -\sum_i \log p_\theta(y_i \mid y_{<i}, O, l, s)
\]

其中 \(O, l, s\) 分别为多视角观察、语言指令、状态，\(y_i\) 包括文本、基础、规划、FAST动作、潜在动作等任何目标令牌。

#### 阶段二：连续动作生成
- 在VLM之后附加一个**DiT（Diffusion Transformer）动作专家**生成连续动作轨迹。
- DiT只使用VLM输出的**潜在动作令牌的K/V缓存**，不访问完整VLM表征。
- 训练DiT时使用**流匹配损失（flow matching loss）**，并通过**梯度解耦**（stop-gradient）防止该损失更新VLM骨干网络：

\[
\mathcal{L}_{\text{FM}} = \mathbb{E}_{A,\epsilon,\tau} \left[ \| v_\phi(A_\tau, \tau, \text{sg}(h_{la})) - (A - \epsilon) \|_2^2 \right]
\]

- 这种设计使得连续控制优化不会破坏VLM已经学到的视觉-语言理解、空间基础和离散动作语义，潜在动作令牌缓存形成稳定的语义桥梁。

### 2.3 语义对齐3D集成
- 在保持VLM语义理解的同时引入3D几何感知。
- 在原2D视觉通路旁**添加一个可训练的3D空间编码器**（选用**VGGT**，利用多视角一致性建模场景布局、物体形状、空间关系）。
- **冻结原始2D视觉编码器**（已通过大规模视觉-语言预训练对齐到LLM表示空间），以保持稳定的语言对齐视觉表示。
- 将2D与3D特征沿特征维度拼接，通过**扩展的视觉投影器**映射到LLM视觉令牌空间：

\[
Z_{\text{vis}} = [ H_{\text{2D}} ; H_{\text{3D}} ] W_{\text{vis}} + b, \quad W_{\text{vis}}^{(0)} = [ W_{\text{Qwen}} ; 0 ]
\]

其中2D部分初始化自原始Qwen2.5-VL投影器，新增的3D部分**零初始化**，保证训练开始时维持原始VLM视觉令牌分布。随着训练进行，3D编码器逐步学习与操作相关的几何结构。

### 2.4 本体规范化
- 目标：使不同机器人的数据在同一个共享表征中学习和迁移，避免将机器人身份注入高层策略表示。
- **动作输出端**：使用**相对末端执行器动作**。将每个未来动作表示为相对于当前末端执行器位姿的SE(3)相对变换：

\[
\Delta T_{t+i} = (T_t^{ee})^{-1} T_{t+i}^{ee}, \quad i = 1,\dots,K
\]

动作块为 \(A_t^{rel} = [\Delta T_{t+1}, \dots, \Delta T_{t+K}, g_{t+1:t+K}]\)，所有未来位姿相对于同一个当前末端位姿，避免累积误差，减少对基坐标的依赖。

- **状态输入端**：使用**末端执行器位姿和关节角度**作为成对状态输入 \(s_t = \{T_t^{ee}, q_t\}\)，其对应关系隐式编码了本体的自由度、关节布局等。对每个机器人本体，仅使用一个**轻量级状态投影器**将其状态映射到共享VLA表示空间。

- **迁移到新机器人时**：采用**两阶段轻量级适配**：先冻结共享VLA骨干，只对齐新状态投影器；再进行轻量端到端微调，最小化对已学操作语义的干扰。

## 6. 主要结论与发现

- GEAR-VLA在**LIBERO上达到98.7%** 平均成功，超越所有对比方法。
- **零样本LIBERO-Plus上达到88.7%**，超越ACoT（86.6%）和π0.5（85.7%），展现出对多种分布变化的鲁棒泛化。

- **通用抓取基准90.1%**，对比π0.5（79.1%）和DexGraspVLA（84.4%），在不规则物体和密集杂乱场景下增益显著。

- **几何感知动作表示**和**本体规范化**是实现强泛化的关键。

## 7. 优点

1. **方法论创新**：粗到细动作学习将离散动作语义与连续动作生成解耦，利用潜在动作令牌作为稳定接口，避免连续损失干扰VLM语义。
2. **3D集成方案巧妙**：冻结2D通路，零初始化3D投影器，渐进式引入几何特征，避免破坏原有语义空间，实现语义对齐。
3. **本体规范化设计简洁有效**：通过相对末端执行器动作和轻量状态投影器将本体差异限制在低级接口，避免共享表示被污染，实现高效跨本体迁移（两阶段适配）。
4. **实验体系全面且具有挑战性**：不仅涵盖标准仿真，还构建了真实世界的双机械臂任务（考虑颜色/尺寸变化）、预训练未见本体迁移、极大规模的通用抓取基准（6360次试验），充分验证了泛化能力。
5. **数据利用高效**：能从无动作标签的人类/网络视频中学习潜在动作动态，减少对昂贵机器人遥操作数据的依赖；在真实任务中仅需200演示，数据效率高（Table 15）。
6. **开源承诺**：代码和模型将发布，有利于社区复现和进一步发展。

## 8. 不足与局限

1. **实验报告缺乏统计分析**：所有成功率报告为单点值，未给出多次运行的均值、标准差或置信区间，难以评估结果稳定性。
2. **真实世界任务类型有限**：仅测试了折叠和抓取等接触丰富操作，未涉及更复杂的组装、插拔等精细操作；且均为桌面操作，缺乏移动操作场景。
3. **本体迁移规模有限**：仅测试了两种双臂机器人（AgileX和LDT-01），未在更多异构本体（如单臂、灵巧手）上验证本体规范化的泛化能力。
4. **依赖多视角和高质量标注**：训练需要多视角RGB图像、状态（末端位姿+关节角）以及部分数据需要FAST令牌和动作标签；通用抓取需要第一帧目标掩膜，虽然文中称只用第一帧，但仍依赖VLM的接地能力与SAM生成掩膜，可能存在失败传播。
5. **算力开销极大**：预训练和策略学习均使用240块H200 GPU，总迭代数超百万，对于资源有限的实验室门槛较高；虽然引入潜在动作ID可从视频学习，但大型预训练本身成本高昂。
6. **潜在动作令牌可能损失部分细节**：因果VQ-VAE以5Hz抽取视频帧，可能丢失快速运动细节；且离散化程度有限（16个码，两组8个），连续动作空间压缩能力有待验证。
7. **消融实验仅在仿真进行**：真实世界的消融（如移除VGGT或潜在动作ID）未在真实机器人上执行，其结论可能不完全推广到真实场景。
8. **未讨论安全性和失败模式**：未分析在未见物体/场景下的失败案例或危险行为，对实际部署的安全考虑不足。
9. **与π0.5比较的公平性**：π0.5为开源模型，但GEAR-VLA使用了不同的VLM基座（Qwen2.5-VL）和更多数据，直接比较可能不完全公平——虽然论文已尽量复制相同协议。
