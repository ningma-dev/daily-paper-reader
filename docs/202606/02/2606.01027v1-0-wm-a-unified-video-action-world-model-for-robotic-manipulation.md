## 摘要
机器人操作需要能够生成可执行动作并在物理执行之前预测和评估其未来后果的模型。我们提出$τ_0$-世界模型（$τ_0$-WM），这是一个统一的视频-动作世界模型，将策略学习、视频预测和动作评估集成在一个单一的未来预测框架中。建立在共享的视频扩散骨干网络上，$τ_0$-WM提供了两个互补的接口。首先，一个视频动作模型从多视角观察、语言指令和机器人状态中联合预测未来的视觉潜在变量和连续动作块。其次，一个动作条件视频模拟器将候选动作块展开为多视角未来，并预测密集的任务进度分数。该模型在大约27,300小时的真实机器人遥操作、UMI风格交互、自我中心人类视频以及使用模态特定监督掩码的回放或失败轨迹上进行训练。在推理时，$τ_0$-WM使用测试时计算来采样动作候选，通过重新去噪一致性对其进行排序，并对低质量候选调用基于模拟器的修正。在具有挑战性的长视野和细粒度机器人操作任务上，$τ_0$-WM显示出优于其他相关基线的性能。

## Abstract
Robotic manipulation requires models that generate executable actions while anticipating and evaluating their future consequences before physical execution. We present $τ_0$-World Model ($τ_0$-WM), a unified video-action world model that integrates policy learning, video prediction, and action evaluation within a single future-predictive framework. Built on a shared video diffusion backbone, $τ_0$-WM provides two complementary interfaces. First, a video action model jointly predicts future visual latents and continuous action chunks from multi-view observations, language instructions, and robot state. Second, an action-conditioned video simulator rolls out candidate action chunks into multi-view futures and predicts dense task-progress scores. The model is trained on approximately $27{,}300$ hours of real-robot teleoperation, UMI-style interaction, egocentric human videos, and rollout or failure trajectories using modality-specific supervision masks. At inference time, $τ_0$-WM uses test-time computation to sample action candidates, rank them with re-denoising consistency, and invoke simulator-based rectification for low-quality candidates. On challenging long-horizon and fine-grained robotic manipulation tasks, $τ_0$-WM shows superior performance over other relevant baselines.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：机器人操作面临“动作执行前无法评估后果”的根本困境。现有方法要么只训练策略（动作生成）而缺乏对未来动态的预测能力，要么将视频预测与策略学习分离，导致无法在物理执行前对动作后果进行推理和修正。
- **背景**：真实机器人遥操作数据昂贵且覆盖场景有限；UMI（通用操控接口）风格数据提供更丰富的视觉交互但动作信号弱；自我中心人类视频提供大规模视觉动态但无机器人兼容动作。如何利用这三种异质数据，训练一个既能生成可执行动作、又能预测和评估未来后果的统一世界模型，是核心挑战。
- **整体含义**：论文提出 τ₀-WM，将策略学习、视频预测和动作评估整合在单一的未来预测框架中，使机器人能够在执行前“想象”并优化动作序列，从而提升长时序、细粒度操作任务的成功率。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- **统一视频-动作世界模型**：基于共享的视频扩散骨干（Wan2.2-TI2V-5B），通过两个互补接口实现“提议-评估-修正”闭环：
  - **视频动作模型（VAM）**：作为策略接口，从多视角观察、语言指令、机器人状态联合预测未来视觉潜变量和连续动作块。
  - **动作条件视频模拟器（ACVS）**：作为评估接口，给定候选动作块，预测多视角未来展开和密集任务进度分数（奖励轨迹）。

### 关键技术细节
1. **架构**：
   - VAM：视频分支（5B参数，Wan DiT骨干） + 动作分支（0.5B参数，DiT风格动作解码器）。动作分支通过交叉注意力耦合视频特征，实现特征级交互。
   - ACVS：复用视频骨干，移除动作分支，但通过将候选动作注入扩散时间嵌入和AdaLN调制嵌入来条件化未来预测。同时增加奖励头预测密集奖励轨迹。
   - 所有分支使用Wan VAE将多视角图像编码为潜变量，并沿空间维度拼接形成统一潜画布。

2. **训练目标**：
   - 对视频潜变量和动作块（或奖励轨迹）均使用流匹配（Flow Matching）目标。
   - **模态特定监督掩码**：异质数据中，真实机器人数据贡献视频+动作监督；UMI数据贡献视频+弱动作监督；自我中心视频仅贡献视频监督；回放/失败轨迹额外贡献奖励监督。每个样本仅对存在的模态计算损失。

3. **训练数据**：约27,300小时，包含：
   - 17.8k小时真实机器人遥操作（AGIBOT-G01、ARX、Franka双机械臂）
   - 6.5k小时UMI风格演示（Gen-DAS夹爪）
   - 3.0k小时自我中心人类交互视频
   - 额外回放/失败轨迹

4. **推理时测试时计算（Test-Time Computation, TTC）**：
   - **阶段1：重新去噪一致性得分（RCS）**：从VAM采样N个候选动作，对每个候选进行随机流时间步去噪，计算平均重新去噪误差，选择一致性最高的候选。
   - **阶段2：低质量动作修正（LAR）**：若RCS得分低于阈值γ，则调用ACVS对所有候选进行视频回放和奖励评估，选出奖励最高（最大任务进度）的候选，然后将其未来潜变量作为额外条件再次查询VAM，生成修正后的动作块。

### 算法流程（文字描述）
```
1. 采样N个候选动作 {a_i}；
2. 计算每个候选的RCS得分 S_i = -平均重新去噪误差；
3. 选择得分最高的候选 a*；
4. 若 S* >= γ，直接执行 a*；
5. 否则，用ACVS评估所有候选，计算奖励轨迹，选出最大进度的候选；
6. 将该候选的未来潜变量转化为未来条件，重新查询VAM，生成修正动作并执行。
```

## 6. 主要结论与发现

1. **统一视频-动作世界模型有效**：τ₀-WM在长时序细粒度操作任务上优于现有策略和视频-动作基线，说明联合预测视觉动态和可执行动作有助于学习更鲁棒的操作行为。
2. **异质预训练带来显著增益**：加入UMI和自我中心数据使零样本成功率从0.14提升至0.55，微调后在杂乱场景下也持续改善，证明异质数据能增强泛化性和鲁棒性。
3. **测试时计算是关键推动力**：RCS+ LAR机制将单次尝试平均成功率从0.43提升至0.60，且优于CFG和ACG等生成引导方法。特别是未来条件修正（LAR）对需要精确定位的任务改善更大。
4. **行为差异**：τ₀-WM倾向于执行额外修正动作（如按压工具），而基线常常“敷衍了事”，推测源于对未来视觉质量的显式建模。

## 7. 优点

- **创新框架**：将策略学习、视频预测和动作评估整合在一个共享骨干中，实现“想象-评估-修正”闭环，而非简单的联合训练。
- **数据高效利用**：通过模态特定监督掩码，首次将真实机器人数据、UMI数据、自我中心视频和失败轨迹统一训练，实现互补监督。
- **测试时计算策略设计巧妙**：先轻量级RCS过滤，再选择性调用重成本ACVS，兼顾效率和鲁棒性。
- **多平台验证**：在三种不同机器人平台上测试，验证框架的泛化性。
- **训练规模大**：~27,300小时数据量级较大，展示了可扩展性。
