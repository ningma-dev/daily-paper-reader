## 摘要
在本报告中，我们介绍Hy-Embodied-0.5-VLA（简称HyVLA-0.5），这是一个覆盖完整机器人学习框架的端到端系统：数据采集、模型设计、持续预训练与监督微调、强化学习后训练以及真实世界部署。该框架中每个组件都扮演着独特的角色。

## Abstract
In this report, we present Hy-Embodied-0.5-VLA, abbreviated as HyVLA-0.5, an end-to-end system that spans the full robot learning stack: data collection, model design, continued pre-training and supervised fine-tuning, RL post-training, and real-world deployment. Each component serves a distinct role in this stack.

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：构建完整的机器人学习堆栈，各组件各司其职：**数据**层面使用定制指尖UMI设备配合动作捕捉系统采集万小时级别的高精度人类演示；**模型**层面扩展Hy-Embodied-0.5骨干网络，结合流匹配动作专家、紧凑记忆编码器和增量块动作表示；**预训练与微调**采用两阶段：先在UMI语料上持续预训练，再通过两个实验轨道（Track-A：目标机器人适应；Track-B：仅使用UMI进行跨实体迁移）进行监督微调；**RL后训练**引入基于近端偏好优化（PRO）的离线RL算法FlowPRO，将失败案例转化为策略改进的驱动力，无需学习奖励模型；**部署**采用异步推理管道和轻量轨迹平滑，实现高频闭环控制。

- **关键技术细节**：
    - **动作表示**：采用“delta-chunk”（增量块）动作表示，预测连续步之间末端执行器的增量运动，解耦策略学习与具体实体运动学，缩小优化空间。
    - **模型架构**：基于Hy-Embodied-0.5-MoT（40亿参数，MoT架构），新增流匹配动作专家（370M参数），采用双塔路由（理解VLM塔+生成动作专家塔），通过共享注意力交互。引入紧凑记忆编码器，利用因果时间空间注意力聚合多帧观测历史，且不增加额外参数（重用ViT的QKV投影）。
    - **流匹配目标**：使用条件流匹配（公式3）直接从噪声预测动作速度场，训练时损失函数为 \( L_{fm} = \mathbb{E} \|v_\theta(A_t^\tau, o_t) - (\epsilon - A_t)\|^2 \)，并混合辅助NTP损失以保留VLM的语义和空间推理能力。
    - **FlowPRO算法**：基于迭代离线RL循环，每轮包括：（1）通过遥操作“干预-回滚”管道收集偏好轨迹对；（2）通过平滑插值将稀疏轨迹级修正转换为密集的逐状态偏好元组；（3）使用RPRO损失（公式8）进行优化，该损失包含对比项和近端正则化项，避免奖励黑客，并利用对比梯度抵消特性安全地与SFT数据共训练。
    - **部署**：平台映射器将相对末端执行器块转换为不同机器人的世界坐标（固定臂用公式10，人形机器人用公式11）；异步推理-执行循环重叠骨干前向与伺服执行；使用三次Bézier曲线进行块拼接，保证C1连续性（公式13-17）。

## 6. 论文的主要结论与发现

1. **全栈设计有效**：HyVLA-0.5在仿真和真实机器人上均达到领先性能。仿真RoboTwin 2.0上Clean 90.9%、Randomized 90.1%，超越π0（+25/+31.7点）和π0.5（+8.2/+13.3点）。
2. **UMI预训练价值**：大尺度高精度UMI预训练能显著提升真实机器人任务成功率，尤其在精度关键的瓶颈步骤上。跨体迁移（Track-B）中，仅使用UMI演示微调即可达到可部署性能，说明UMI提供了实体无关的动作先验。
3. **FlowPRO有效**：基于PRO的离线RL方法优于DAgger和π0.6*，在4个任务上均达到最高成功率和最短完成时间（Bottle 99%/16s，Cap 99%/21s，USB 98%/22s，Zip 94%/37s）。FlowPRO将失败案例转化为策略改进，无需奖励模型，且通过近端正则化避免奖励黑客。
4. **紧凑记忆编码器带来增益**：在仿真消融中，移除记忆编码器导致性能下降（~2点），进一步移除UMI预训练再降~0.7点。
5. **部署工程贡献**：异步推理和Bézier平滑能够实现高频（50Hz）闭环控制，保证了策略在真实硬件上的可用性。

## 7. 优点：方法或实验设计上的亮点

- **方法亮点**：
    - **数据采集创新**：指尖UMI配合动作捕捉，兼顾亚毫米精度与自然触觉反馈，且动作捕捉数据可直接用于RL后训练，减少目标机器人数据需求。
    - **增量块（delta-chunk）表示**：解耦策略与特定运动学，便于跨体迁移。
    - **FlowPRO**：无批评/无奖励的离线RL，利用失败样本直接优化，规避了奖励设计瓶颈，且具有对比梯度抵消特性，可安全混合SFT数据。
    - **部署设计**：训练无关的Bézier块拼接，保证C1连续性，适用于任意策略。
    - **紧凑记忆编码器**：无额外参数，兼容单帧ViT初始化，降低计算负担。

- **实验设计亮点**：
    - 明确分离Track-A（同体适应）和Track-B（跨体迁移），系统评估UMI预训练的跨体泛化能力。
    - FlowPRO实验采用多种子、多迭代、统计报告标准差，具有良好的可重复性。
    - 消融实验（记忆编码器、UMI预训练）设计简洁，验证关键组件贡献。
