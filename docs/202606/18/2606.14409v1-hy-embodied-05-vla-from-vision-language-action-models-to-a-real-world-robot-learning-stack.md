在本报告中，我们介绍了Hy-Embodied-0.5-VLA（简称HyVLA-0.5），这是一个端到端系统，涵盖完整的机器人学习栈：数据收集、模型设计、持续预训练和监督微调、强化学习后训练以及真实世界部署。该栈中的每个组件都扮演着不同的角色。

## Abstract
In this report, we present Hy-Embodied-0.5-VLA, abbreviated as HyVLA-0.5, an end-to-end system that spans the full robot learning stack: data collection, model design, continued pre-training and supervised fine-tuning, RL post-training, and real-world deployment. Each component serves a distinct role in this stack.

## 论文核心问题与整体含义（研究动机和背景）

论文指出，通用机器人不能仅靠单一模型实现，而是需要构建完整的机器人学习栈，涵盖数据收集、模型设计、训练、微调、强化学习后训练以及真实部署。现有方法在以下方面存在关键瓶颈：

- **数据侧**：传统遥操作不自然、缺乏触觉反馈；人类数据粗粒度、缺乏精准动作标签；跨实体迁移面临动作空间、动力学和感知的异构性。
- **架构侧**：早期使用离散化动作令牌导致精度和频率受限；通用VLM未针对机器人控制优化，缺乏密集时空推理能力。
- **训练侧**：标准模仿学习难以达到灵巧操作；强化学习依赖脆弱的奖励或价值模型，设计复杂。
- **部署侧**：高频率闭环控制需求常被忽视，政策推理延迟与伺服控制不匹配。

HyVLA-0.5 旨在通过统一的全栈方法同时解决这些耦合问题，将数据采集、模型设计、预训练/微调、RL后训练和部署作为协同设计的组件，从而实现可部署的通用机器人。

## 论文提出的方法论

### 核心思想
构建端到端VLA系统，每个阶段针对具体瓶颈设计专用组件，使策略学习与实体运动学解耦，并在真实硬件上实现高频闭环控制。

### 关键技术细节

- **数据收集**：定制指尖UMI装置，配合外部光学动捕笼，收集超过10k小时（100万+episodes）的自我中心、亚毫米精度人类示教数据。指尖设计保留自然触觉，动捕系统替代SLAM提供高保真轨道，同一轨迹可直接用于后训练，减少目标机器人数据采集需求。

- **模型架构**：
  - 骨干：基于Hy-Embodied-0.5-MoT（4B参数，混合Transformer架构），原生分辨率视觉编码，模态自适应计算。
  - 动作专家：双塔流匹配模型，连续动作预测（delta-chunk表示），与VLM通过共享自注意交互。使用块式因果注意保持VLM预训练分布。
  - 紧凑记忆编码器：在ViT中每4层插入因子化时空注意力（因果时序+空间双向），历史帧信息压缩到当前帧token，不增加VLM视觉令牌数量，K=1时可退化为单帧编码器。
  - **Delta-chunk动作表示**：动作定义为相对于当前末端执行器位姿的增量（平移+6D旋转+夹爪），解耦策略学习与特定实体运动学，缩小优化空间。

- **预训练与微调**：
  - 在Hy-UMI-10K数据集上预训练200k步（全局batch 1024），仅流匹配目标，K=1。
  - 监督微调：分两条轨道。Track-A（同实体适应）在目标机器人遥操作数据上微调；Track-B（仅UMI跨实体迁移）仅使用任务特定UMI示教微调，部署到形态不同的机器人（JAKA K1、Astribot S1）。微调时K=6，使用历史5帧。

- **RL后训练（FlowPRO）**：
  - 无需奖励模型或批评器，基于偏好优化。通过“干预-回滚”数据流水线，一对多状态偏好对（$(s, a_w, a_l)$）通过平滑插值合成。
  - RPRO损失（公式8-9）：对比项（拉近 $a_w$、推远 $a_l$）加对称近端正则化项（锚定隐式奖励幅度，防止奖励黑客）。同时加入SFT项（公式9）保持基础性能。
  - 利用对比梯度抵消性质：当 $a_w = a_l$ 时，对比梯度为零，安全混合SFT样本。

- **部署**：
  - 异步推理管线：推理线程与执行线程重叠，隐藏骨干延迟。
  - 贝塞尔平滑器：三次C1连续曲线连接新旧动作块，利用历史与未来切线方向，实现平滑过渡。
  - 实体映射：固定基座臂直接重投影到世界坐标系；人形机器人通过启发式推断躯干位姿（算法1），保持动作接口不变。

## 主要结论与发现

1. HyVLA-0.5 在模拟基准上达到最优（Clean: 90.9%, Randomized: 90.1%），显著超越 π0 和 π0.5。
2. 紧凑记忆编码器在模拟中贡献约2%的提升；UMI预训练在模拟中增益有限（因域差异大），但在真实世界中大幅提升精度关键步骤成功率。
3. FlowPRO 无需奖励模型，通过偏好优化持续提升策略成功率，超越 DAgger 和 π0.6*，完成时间也更短。
4. UMI 预训练提供实体无关的动作先验，使仅用UMI示教即能成功迁移到形态不同的真实机器人（JAKA K1, Astribot S1）。
5. UMI收集的力信号可用于策略学习，实现触觉辨别任务。

## 优点
