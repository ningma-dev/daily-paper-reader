## 摘要
人类自然地利用多种感知模态与物理世界交互，而大多数用于机器人的视觉-语言-动作（VLA）模型仅依赖RGB观测。这限制了它们感知那些难以或无法从RGB相机推断的物理属性的能力，例如温度、声音或雷达响应。我们提出MuseVLA，一种自适应多模态感知VLA模型，它将新型传感器整合为机器人操作的按需工具。给定任务指令和视觉上下文，MuseVLA首先生成一个传感器令牌和目标描述，选择要调用的感知模态以及关注对象，类似于带参数的工具调用。然后，它将选定的传感器测量值转换为接地传感器图像，这是一种统一的中间表示，编码异构读数用于多模态融合和动作生成。这种设计将特定传感器的处理与VLA主干解耦，实现了多种模态的高效集成。为了减少对昂贵的多感官机器人数据集的需求，我们进一步引入了一个数据合成流水线，用接地传感器图像扩充现有的RGB视频数据集，从而能够泛化到未见过的传感器引导任务。我们在真实机器人上评估了MuseVLA，涉及需要多模态感知输入的挑战性灵巧手操作任务，包括温度引导的拾取和放置、音频驱动的物体搜索以及雷达辅助的隐藏物体检索。MuseVLA平均成功率达到80.6%，显著优于仅RGB和多感官VLA基线，并在未见任务上表现出强大的零样本能力。

## Abstract
Humans naturally leverage diverse sensing modalities to interact with the physical world, while most Vision-Language-Action (VLA) models for robotics rely solely on RGB observations. This limits their ability to perceive physical properties that are difficult or impossible to infer from RGB cameras, such as temperature, sound, or radar response. We present MuseVLA, an adaptive multimodal sensing VLA model that integrates novel sensors as on-demand tools for robotic manipulation. Given a task instruction and visual context, MuseVLA first generates a sensor token and target description that select the sensing modality to invoke and what to attend to, analogous to a tool call with arguments. It then converts the selected sensor measurement into a grounded sensor image, a unified intermediate representation that encodes heterogeneous readings for multimodal fusion and action generation. This design decouples sensor-specific processing from the VLA backbone, enabling efficient integration of diverse modalities. To reduce the need for expensive multisensory robot datasets, we further introduce a data synthesis pipeline that augments existing RGB video datasets with grounded sensor images, enabling generalization to unseen sensor-guided tasks. We evaluate MuseVLA on a real-world robot across challenging dexterous hand manipulation tasks that require multimodal sensing inputs, including temperature-guided pick-and-place, audio-driven object search, and radar-assisted hidden object retrieval. MuseVLA achieves 80.6% success rate on average, outperforming RGB-only and multisensory VLA baselines significantly, and exhibits strong zero-shot capabilities on unseen tasks.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 将传感器视为“按需工具”，模型根据任务指令和 RGB 图像动态选择最相关的传感器并生成目标描述，然后将选中的传感器测量值转换为统一的**接地传感器图像**（grounded sensor image），与 RGB 图像一起用于动作生成。该设计将传感器特定处理与 VLA 主干解耦，可灵活扩展新传感器。

### 关键技术细节
1. **传感器令牌（Sensor Token）与自适应选择**：
   - 引入可学习的传感器令牌：`<None>`（仅 RGB）、`<Thermal>`（热感）、`<Acoustic>`（声学）、`<mmWave>`（毫米波雷达）。模型根据任务指令和 RGB 图像生成对应的传感器令牌和目标描述（如“hot drink”），类似带参数的工具调用。
2. **接地传感器图像（Grounded Sensor Image）**：
   - 将选中的传感器原始数据（如热力图、声强图、雷达反射图）通过空间对齐投影到 RGB 图像平面，并利用 SAM3 分割模型将目标区域掩码与传感器热力图叠加在 RGB 图像上，形成统一的 RGB 空间表征。公式如下：
     ```
     M = f_seg(o_RGB, l_d)       // 从RGB和描述得到掩码
     m = M ⊙ s + (1 - M) ⊙ o_RGB // 将热力图覆盖在掩码区域
     ```
   - 这样所有传感器数据都转换为类似 RGB 图像的格式，复用同一视觉编码器。
3. **动作生成**：
   - 使用预训练 VLM（PaliGemma-2）作为主干，结合扩散 Transformer（DiT）动作专家。将接地传感器图像编码后与任务指令、RGB 特征、机器人状态一起输入，通过扩散过程生成动作序列。
4. **数据合成管线**：
   - 构建传感器字典（如 thermal: hot/cold/warm; acoustic: ringing/quiet; radar: occupied/empty），从现有 RGB 机器人数据集（如 MolmoAct、AgiBotWorld-Alpha、VITRA）中随机采样，将物理关键词注入任务指令，并通过 VLM 生成目标描述，SAM3 分割目标区域后叠加颜色掩码模拟传感器观测，从而生成大规模多感官训练数据（共 9.6K 片段、105 万帧）。

### 训练损失
- 联合优化 VLM 损失（传感器令牌 + 目标描述的交叉熵）和扩散 MSE 损失：
  ```
  L = L_VLM + λ * E[||ε - ε_θ(a_τ, τ, c)||²]
  ```
  其中 λ=1e-2。

## 7. 优点

- **创新性设计**：传感器令牌 + 接地传感器图像解耦了传感器处理与 VLA 主干，使新传感器可无缝集成无需重新训练整个模型。
- **数据高效**：无需大量真实多传感器数据，通过合成 RGB 数据即可训练，显著降低数据成本。
- **统一表征**：所有传感器数据转换为 RGB 格式，复用视觉编码器，减少模型复杂度。
- **自适应与高效**：模型按需选择传感器，避免冗余计算，推理更快、内存更低。
- **强泛化性**：在零样本任务上表现优异，证明合成数据可跨场景迁移。
- **实验充分**：包含消融、基线对比、显著性检验、多阶段任务，且代码和数据将开源。

## 8. 不足与局限

- **数据集规模有限**：真实世界数据集仅 720 个遥操作片段，未覆盖大量物体和复杂场景，可能限制模型在更广泛环境中的泛化。
- **依赖分割模块**：接地传感器图像的构建依赖于 SAM3 的分割质量，若分割失败（如目标遮挡、罕见物体），则传感器信息可能被错误叠加，影响性能。
- **传感器校准简化**：空间对齐通过一次性单应矩阵近似，适用于固定工作距离，若机器人基座移动或视角变化，需重新校准。
- **未涉及动态环境**：所有任务均为静态场景，未考虑移动平台或动态障碍物场景。
- **任务类型覆盖有限**：仅包括拾放、搜索、开盒等灵巧手操作，未评估更为复杂的长序列或接触力敏感任务。
- **对比基线较少**：主要与 π0/π0.5 比较，未与近期其他多模态 VLA（如 OmniVLA、Tactile-VLA）进行直接对比（可能因发表时间限制）。
- **合成数据真实性存疑**：使用颜色掩码模拟传感器数据，虽能辅助训练，但与真实传感器物理特性（如热噪声、镜面反射）存在差异，可能影响真实场景性能。
