## 摘要
在现实世界中实现鲁棒的机器人操作，不仅需要理解当前的观测，还需要具备记忆和动力学建模能力。世界动作模型（WAM）通过联合建模视觉预测和基于当前及历史观测的动作来具备这些能力，使其成为机器人操作的一个有前景的范式。然而，现有的WAM面临一个基本的权衡：具有高效推理的方法通常仅以最近观测的有界窗口为条件，因此在非马尔可夫环境中表现不佳；而保留长历史的方法则随着序列长度增长，时间和空间成本大幅增加。为了解决这一挑战，我们引入了MemoryWAM，一种具有高效持久记忆的世界动作模型。MemoryWAM采用混合记忆设计，结合了最近帧、事件边界锚定帧以及总结长程历史的紧凑要旨标记。一种定制的注意力机制能够检索详细的短期上下文和压缩的长期上下文，支持依赖记忆的决策制定，同时降低推理延迟和GPU内存使用。在模拟和现实世界的长周期、依赖记忆的操作任务中，MemoryWAM在保持良好计算效率的同时，优于强大的视觉-语言-动作（VLA）和WAM基线。

## Abstract
Robust robotic manipulation in the real world requires not only an understanding of the current observation, but also memory and dynamics modeling. World action models (WAMs) possess these capabilities by jointly modeling visual foresight and actions conditioned on both current and historical observations, making them a promising paradigm for robotic manipulation. However, existing WAMs face a fundamental trade-off: methods with efficient inference typically condition only on a bounded window of recent observations and therefore struggle in non-Markovian environments, whereas methods that preserve long histories incur time and space costs that grow substantially with sequence length. To address this challenge, we introduce MemoryWAM, a world action model with efficient persistent memory. MemoryWAM uses a hybrid memory design that combines recent frames, event-boundary anchor frames, and compact gist tokens that summarize long-range history. A tailored attention mechanism enables retrieval of both detailed short-term context and compressed long-term context, supporting memory-dependent decision-making with reduced inference latency and GPU memory usage. Across long-horizon, memory-dependent manipulation tasks in both simulation and the real world, MemoryWAM outperforms strong vision-language-action (VLA) and WAM baselines while maintaining favorable computational efficiency.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：现实中的机器人操作任务需要同时理解当前观测、记忆历史信息并建模物理动态。世界动作模型（World Action Models, WAMs）通过联合建模视觉预测和动作来具备这些能力，但现有方法面临一个根本性的权衡：
  - 高效推理的方法（如滑动窗口）仅依赖有限最近的观测，在非马尔可夫环境中因无法访问长程信息而失败。
  - 保留全部历史的方法（如完整KV缓存）虽然保留上下文，但推理延迟和GPU内存随序列长度线性增长，难以实用。
- **研究动机**：受认知心理学中人脑混合记忆系统（短期记忆、长期摘要记忆、事件边界记忆）的启发，设计一种高效的持久记忆机制，使WAM既能保持长程上下文，又不显著增加计算成本。

## 2. 提出的方法论：核心思想、关键技术细节
- **核心思想**：提出MemoryWAM，一种混合记忆设计，包含三种互补的记忆组件：
  - **短期记忆**：滑动窗口缓存最近 N_recent 帧的完整视觉token，用于即时闭环控制（保持高保真局部上下文）。
  - **事件边界记忆**：保留任务开始时的少量锚定帧（anchor frames）原始视觉token，因为初始场景配置往往包含指令关键信息，后来可能被遮挡或滑出窗口外。
  - **要旨记忆（Gist Memory）**：为每一帧附加 M 个可学习的要旨token（gist tokens），其中 M << L（L是每帧视觉token数）。对于既不是锚帧也不是最近帧的历史帧，后续的视频/动作token不再直接关注其原始视觉token，而是关注其要旨token，从而形成长程历史的压缩表示。推理时，这些历史帧的完整KV缓存被清除，仅保留要旨token的KV缓存，实现压缩比 d = L/M，将长程缓存复杂度从 O(N) 降低到 O(N/d)。
- **关键技术细节**：
  - **模型架构**：基于预训练Wan2.2-TI2V-5B，使用混合Transformer（MoT）架构：视频DiT（约5B参数）提取动态感知特征并维护记忆；动作DiT（约1B参数，隐藏维度1024）生成动作块，通过交叉注意力访问视频DiT的KV缓存。
  - **训练协议**：采用连续流匹配（flow matching），视频和动作分支共享视频DiT的3D RoPE位置编码。训练时对每个episode构建自回归序列，应用混合记忆注意力掩码，使得训练时的可见性与推理时的KV缓存一致。对清洁条件潜在帧进行随机噪声增强（self-forcing），防止过拟合。
  - **推理协议**：动作DiT使用50步（仿真）或10步（真实）去噪生成动作块，视频生成被禁用（利用WAM“训练时监督视频预测，推理时仅需前向传播”的优势）。每执行完16个动作后，从动作轨迹中采样4帧更新观测缓冲区，编码为新条件帧。

- 结论：混合记忆设计不仅是效率妥协，更是有效的记忆结构，能平衡短期细节、事件边界和长期抽象。
