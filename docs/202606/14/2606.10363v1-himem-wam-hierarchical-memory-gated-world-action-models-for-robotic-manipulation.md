## 摘要
世界动作模型（WAMs）已成为具身智能的一种新的强大范式，学习与动作相关的视觉动态，显著提升了泛化能力和鲁棒性。然而，现有的WAMs在长期机器人操作中仍然难以处理与任务相关的记忆。为了解决这个问题，我们提出了HiMem-WAM，一种分层记忆门控WAM，它整合了以运动为中心的潜在动作、高层技能潜在变量和边界触发的记忆更新。具体来说，我们开发了一个分层潜在动作框架，联合学习低层运动和高层技能潜在变量，提供结构化的时间抽象。同时，一个边界感知的记忆门在预测的技能转换处写入紧凑的任务状态，使得无需在测试时生成未来视频或光流估计即可进行因果推理。在LIBERO、LIBERO-PLUS、RMBench和真实世界任务上的评估表明，HiMem-WAM的分层潜在变量提高了部署扰动下的鲁棒性，并且记忆模块极大地有益于依赖记忆的长期操作。

## Abstract
World Action Models (WAMs) have emerged as a new powerful paradigm for embodied intelligence, learning action-relevant visual dynamics that significantly enhance generalization and robustness. However, existing WAMs still struggle with task-relevant memory in long-horizon robotic manipulation. To address this, we present HiMem-WAM, a Hierarchical Memory-Gated WAM that integrates motion-centric latent actions, high-level skill latents, and boundary-triggered memory updates. Specifically, we develop a hierarchical latent action framework that jointly learns low-level motion and high-level skill latents, providing structured temporal abstraction. Meanwhile, a boundary-aware memory gate writes compact task states at predicted skill transitions, enabling causal inference without test-time generation of future video or optical flow estimation. Evaluated on LIBERO, LIBERO-PLUS, RMBench and real-world tasks, HiMem-WAM shows that hierarchical latents improve robustness under deployment perturbations, and the memory module substantially benefits memory-dependent long-horizon manipulation.

# HiMem-WAM 论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）

**问题**：现有世界动作模型（WAMs）在长程机器人操作中缺乏有效的任务相关记忆机制。尽管 WAMs 通过学习与动作相关的视觉动态提升了泛化性和鲁棒性，但在需要依赖历史状态的长期任务中表现不足，例如需要回忆被遮挡的子任务状态或适应部分完成后的任务继续。

**意义**：将低层运动抽象为可复用的高层技能，并显式保留跨技能边界的任务记忆，是提升 WAMs 在复杂长程操作中实用性的关键。HiMem-WAM 提出了一种分层记忆门控框架，旨在同时解决动作抽象和记忆保持两个核心挑战。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 将策略分解为三个角色：**高层技能选择**（规划器）、**低层运动展开**（执行器）和 **动作落地**（解码器）。
- 引入**记忆门控模块**，仅在预测的技能边界处写入紧凑任务状态，实现因果推理（无需测试时生成未来视频或光流）。

### 关键技术细节

**（a）低层潜在动作（Low-level Latent Actions）**
- 使用多视角光流（DPFlow）作为运动监督，训练一个变分分词器（Variational Tokenizer），将短期上下文（当前/下一帧观测、光流、本体感觉、语言指令）编码为低层潜在变量 \(z^l_t\)。
- 训练目标：重构光流 + 弱对齐真实动作（若有动作标签）+ KL散度正则化。
- 分词器冻结后离线提取所有轨迹的低层潜在序列 \(Z^l\)。

**（b）高层技能潜在（High-level Skill Latents）**
- 通过分层深度网络自动发现技能边界：对低层潜在序列进行多级分段，每级使用边界评分（基于查询-键的差异度）和注意力池化，生成更高层技能序列 \(Z^h\)。
- 训练损失包括：下一潜在预测损失、运动重构损失、边界比率正则化和段内一致性损失。

**（c）记忆门控模块**
- **读门**（Read Gate）：从外部记忆库 \(M_t\) 中检索历史上下文 \(c^m_t\)，通过门控 \( \alpha^r_t = \sigma(G_r(\cdot))\) 融合到当前状态 \(\tilde{x}_t\)。
- **写门**（Write Gate）：使用规划器预测的边界分数 \( \hat{b}_t\) 和门控 \( \alpha^w_t\)，当 \(\alpha^w_t > \eta\) 时写入新记忆 token \(\gamma_t\)，写入监督来自真实边界标签。
- 记忆更新由预测的边界触发，避免密集历史聚合，保持因果可逆。

**（d）三阶段训练流程**
- **Stage I**：训练低层潜在动作分词器。
- **Stage II**：冻结分词器，发现技能边界，预训练规划器和执行器（无外部记忆）。
- **Stage III**：激活记忆门控，在带动作标签的数据上微调完整策略。

**推理过程**（因果）：当前观测 + 记忆 → 读记忆 → 规划器预测技能和边界分数 → 执行器展开低层潜在动作块 → 解码为可执行控制 → 若写门打开则写入新记忆。
