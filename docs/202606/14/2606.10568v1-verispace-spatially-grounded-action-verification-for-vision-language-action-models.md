## 摘要
视觉-语言-动作（VLA）模型在机器人操作中展现出巨大潜力，但其在测试时的可靠性仍受限于一次性动作预测，即便是微小的动作误差也可能导致抓取失败、碰撞或任务过程错误。一个自然的替代方案是为VLA系统配备测试时验证机制，允许在执行前提出并评估多个候选动作。然而，可靠的动作验证颇具挑战，因为它不仅需要区分候选动作之间的细微几何差异，还需评估动作是否朝任务目标取得有意义的进展。我们提出VeriSpace，一种面向VLA系统中测试时动作选择的3D感知动作验证器。VeriSpace通过两个关键组件评估候选动作：双路径3D注入场景编码（构建同时保留视觉语义和显式3D几何的场景表示）和空间基础动作推理（通过推理任务相关的空间关系、几何有效性和预期目标进展来评估每个动作）。这些组件共同作用，使得在保持与现有VLA策略完全兼容的同时，能够更可靠地区分细微但结果关键的候选动作。在公开基准和真实机器人操作任务上的实验表明，VeriSpace在分布内和分布外场景中均能持续提升基础VLA策略及先前基于验证方法的决策可靠性，带来显著收益。

## Abstract
Vision-language-action (VLA) models have shown strong promise for robotic manipulation, but their reliability at test time remains limited by one-shot action prediction, where even small action errors can cause grasp failure, collision, or incorrect task progression. A natural alternative is to equip VLA systems with test-time verification, allowing multiple candidate actions to be proposed and evaluated before execution. However, reliable action verification is challenging because it requires not only distinguishing subtle geometric differences between candidate actions, but also assessing whether an action makes meaningful progress toward the task goal. We present VeriSpace, a 3D-aware action verifier for test-time action selection in VLA systems. VeriSpace evaluates candidate actions through two key components: Dual-Path 3D-Injected Scene Encoding, which constructs a scene representation that jointly preserves visual semantics and explicit 3D geometry, and Spatially-Grounded Action Reasoning, which evaluates each action by reasoning over task-relevant spatial relations, geometric validity, and expected goal progress. Together, these components enable more reliable discrimination between subtle yet outcome-critical action candidates while remaining fully compatible with existing VLA policies. Experiments on public benchmarks and real-world robotic manipulation tasks show that VeriSpace consistently improves decision reliability over both underlying VLA policies and prior verification-based methods, yielding substantial gains in both in-distribution and out-of-distribution settings.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：VeriSpace是一个3D感知的动作验证器，接收当前RGB-D观测（静态图像+深度图）和任务指令，对每个候选动作进行结构化空间推理，并输出评估分数，用于重排序选择。
- **关键技术细节**：
  - **双路径3D注入场景编码（Dual-Path 3D-Injected Scene Encoding）**：
    - 利用深度图反投影得到3D坐标图，使用正弦编码+MLP生成3D几何嵌入。
    - 构建两个互补的token流：
      - *3D几何token*：直接将几何嵌入投影到VLM token空间，提供显式几何结构。
      - *空间增强视觉token*：将3D几何嵌入与CLIP视觉特征相加，保留语义且获得3D感知。
    - 进一步通过**几何引导局部聚合（GLA）**，使用最远点采样（FPS）选取种子token，在3D球邻域内聚合特征，捕捉局部几何结构（如边缘、间隙）。
  - **空间基础动作推理（Spatially-Grounded Action Reasoning）**：
    - 采用链式思维（CoT）进行三步推理：
      1. **动作空间推理**：分析任务相关实体（物体、机器人末端）的精细空间关系（位置、方向、接触、间隙）。
      2. **目标进展评估**：对比当前状态与预期执行后状态，判断动作是否使场景更接近目标。
      3. **验证分数估计**：综合推理结果输出标量分数。
  - **学习目标**：
    - 动作评估损失（Bradley-Terry偏好对）：使验证器对更好动作赋予更高分数。
    - 推理损失（交叉熵）：监督CoT序列生成。

## 6. 主要结论与发现

- VeriSpace在**所有设置**下均显著提升基础VLA策略的成功率：

- VeriSpace作为**即插即用模块**，可兼容多种VLA架构，训练后无需修改原策略。

## 7. 优点

1. **方法创新**：首次将**显式3D几何感知**引入VLA测试时验证，通过双路径编码和局部聚合精准捕获空间细节。
2. **结构化推理**：使用链式思维（CoT）逐步进行空间推理和目标进展评估，优于直接评分方法。
3. **实用性强**：即插即用，不改变原VLA策略的训练或结构；推理开销通过并行化控制在可接受范围。
4. **泛化性验证充分**：涵盖模拟器（SimplerEnv, LIBERO）和真实机器人、多种VLA backbone、分布内与分布外任务。
5. **消融全面**：逐一剖析各组件贡献，提供了清晰的收益来源分析。

## 8. 不足与局限

1. **对3D输入的依赖**：需要深度图（或深度估计模型），在无深度传感器或估计不准的场景性能可能下降。
2. **长时任务提升有限**：在LIBERO-Long子集上，虽然仍有提升（+0.6 pp和+7.1 pp），但幅度相对较小，说明长序列推理仍是挑战。
3. **CoT数据构建复杂**：依赖多个外部模型（深度估计、目标检测、空间推理VLM）进行合成，可能引入噪声或误差。
4. **计算开销**：尽管已优化，但k个候选动作仍需k次推理，实时性在高频控制场景可能受限。
5. **未与端到端3D VLA（如SpatialVLA）深入比较**：仅在SimplerEnv上对比了3D-VLA（表7），但未讨论为何不直接训练此类模型。
6. **实验覆盖**：未涉及双臂操作或动态场景（如移动物体），真实世界任务数量相对较少（6个）。
