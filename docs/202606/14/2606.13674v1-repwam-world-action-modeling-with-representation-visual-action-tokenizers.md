## 摘要
本文提出了RepWAM，一种以表示为中心的世界动作模型，构建在表示视觉-动作分词器之上。现有的世界动作模型通常从预训练的视频生成模型中继承面向重建的视频分词器。尽管这些分词器保持了视觉保真度，但仅靠像素重建对于学习连接未来预测与机器人控制的指令跟随动态所提供的指导有限。为解决这一问题，我们探索了一种用于以表示为中心的世界动作建模的语义视觉-动作潜在空间。具体而言，我们训练了一个表示视觉-动作分词器，将视觉输入映射为对齐的视觉和潜在动作令牌。然后，我们预训练世界动作模型，以联合建模未来的视觉状态以及连接它们的语言指令下的潜在动作，随后适应真实机器人轨迹以实现闭环操作。在真实世界操作任务和仿真基准上的实验表明，RepWAM在各种操作场景中均展现出强大性能，而消融实验则凸显了语义视觉-动作分词相较于面向重建的替代方案的价值。这些结果确立了表示视觉-动作分词作为世界动作模型的一个有前途的基础，并朝着通用机器人策略迈出了一步。代码和权重将在 https://github.com/wdrink/RepWAM 提供。

## Abstract
This work presents RepWAM, a representation-centric world action model (WAM) built on representation visual-action tokenizers. Existing WAMs typically inherit reconstruction-oriented video tokenizers from pretrained video generation models. Although these tokenizers preserve visual fidelity, pixel reconstruction alone provides limited guidance for learning instruction-following dynamics that connect future prediction with robot control. To address this, we explore a semantic visual-action latent space for representation-centric world action modeling. Specifically, we train a representation visual-action tokenizer that maps visual inputs into aligned visual and latent action tokens. We then pretrain our WAM to jointly model future visual states and the latent actions that connect them under language instructions, followed by adaptation to real robot trajectories for closed-loop manipulation. Experiments on real-world manipulation tasks and simulation benchmarks show that RepWAM delivers strong performance across diverse manipulation settings, while ablations highlight the value of semantic visual-action tokenization over reconstruction-oriented alternatives. These results establish representation visual-action tokenization as a promising foundation for world action models and a step toward generalist robot policies. Code and weights will be available at https://github.com/wdrink/RepWAM.

## 1. 论文的核心问题与整体含义（研究动机和背景）

现有世界动作模型（WAM）通常从预训练的视频生成模型中继承面向像素重建的视频分词器（tokenizer）。这些分词器虽然能保持视觉保真度，但像素重建本身无法有效学习连接未来预测与机器人控制的**指令跟随动态**。具体缺陷体现在两方面：
- **视觉侧**：重建驱动的分词器将潜在容量浪费在背景纹理等低级外观上，而物体身份、空间关系、交互线索等操作相关语义被弱化，导致语言条件的世界模型难以正确建模目标导向的动态。
- **动作侧**：视觉潜在编码与机器人动作指令处于不同空间，迫使每一步都需要一个逆动力学模型来桥接模态差距。

**核心问题**：世界动作模型到底需要什么样的表示？能否设计一个共享的语义潜在空间，使得视觉状态和它们之间的动作过渡被统一表示？

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
提出**RepWAM**（Representation World Action Model），一种以表示为中心的世界动作模型，建立在**表示视觉-动作分词器（RepViTok）**之上。该分词器将视觉输入映射为与视觉基础模型对齐的语义视觉令牌，并在此空间内诱导出潜在动作令牌。

### 关键技术细节

#### (1) 表示视觉-动作分词器（RepViTok）
- **视觉分词**：使用ViT自动编码器，将首帧按16x16分块，后续帧按4x16x16时-空管状分块。编码器采用时间因果掩码+全空间注意力的Transformer。训练时除重建损失（L1、感知损失、对抗损失）外，还加入**特征对齐损失**：将视频潜在编码平均池化后与冻结的视觉基础模型（Perception Encoder）输出的平均池化特征进行L2对齐，引导潜在空间承载语义信息。总损失：ℒ_vis = ℒ_rec + λ_align ℒ_align。
- **潜在动作分词**：在冻结的视觉分词器之上，训练耦合的**逆动力学模型（IDM）**和**前向动力学模型（FDM）**。IDM将连续两帧视觉令牌 `(z_t, z_{t+1})` 压缩成低维潜在动作 ℓ_t（d_ℓ=4）；FDM利用 ℓ_t 生成一个软传输矩阵 K_t 和残差 δ_t，通过 `ẑ_{t+1} = K_t z_t + δ_t` 重建下一帧。训练使用前向预测损失和反向一致性损失。

#### (2) 因果世界动作模型（Causal WAM）
- 采用**因果扩散Transformer**，将语言指令编码为条件令牌。对视觉和动作令牌按时间窗口分块（chunk size = k），使用块级因果掩码。每个块内视觉和动作令牌共享注意力权重但使用模态特定的前馈网络。
- 使用**条件流匹配（Conditional Flow Matching）**目标训练：对每个块采样噪声和时间标量α，构造线性插值x_α，让网络预测速度 `ẋ_α = u - ε`。损失为视觉部分和动作部分速度预测误差的加权和（λ_a=1）。训练采用教师强制。

#### (3) 两阶段适应
- **第一阶段**：在大规模机器人视频数据（AgiBot）上预训练WAM，预测视觉和潜在动作令牌。
- **第二阶段**：在混合真实机器人轨迹数据（AgiBot、RoboMIND、RoboCOIN、InternA1）上微调，将潜在动作映射到实际机器人动作，实现闭环操作。

1. **语义视觉-动作对齐显著提升世界动作建模**：RepViTok相比重建导向分词器（WAN2.2 VAE、ViTok）在视觉动态预测和动作预测上均取得更好结果（gFVD、OLS），并在真实闭环操作中成功率更高。
