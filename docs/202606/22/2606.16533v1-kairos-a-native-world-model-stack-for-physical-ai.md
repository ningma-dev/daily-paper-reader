## 摘要
世界模型正在从被动的视觉生成器转变为物理AI的基础性、可操作基础设施：它们必须原生地从异构经验中获取世界知识，在长时间跨度内维持持久状态，并在实际部署约束下高效执行。我们提出了Kairos，一个围绕这些需求设计的原生世界模型栈。（1）Kairos通过首创的、由跨具身数据课程引导的原生预训练范式来学习世界，该课程将开放世界视频、人类行为数据和机器人交互组织成一个渐进的发展路径。（2）Kairos通过配备混合线性时间注意力的原生统一架构来维持世界，实现统一的世界理解、生成和预测，其中滑动窗口注意力捕获局部动态，扩张滑动窗口捕获中程依赖，门控线性注意力维持持久的全局记忆。我们建立了严格的理论界限，证明这种时间分解严格限制了误差累积，从数学上保证了长时程的状态传播。（3）Kairos通过引入部署感知的系统协同设计来运行世界，支持在服务器和消费级硬件上进行低延迟的推出生成，用于真实世界的观察-行动-反馈循环。在具身世界模型、长时程和行动策略基准上的实验表明，Kairos在实现顶级性能的同时，提供了强大的效率-能力权衡。这些结果共同将Kairos定位为未来自进化物理智能的凝聚性操作基础。

## Abstract
World models are transitioning from passive visual generators to foundational, operational infrastructure for Physical AI: they must natively acquire world knowledge from heterogeneous experience, maintain persistent states over long horizons, and execute efficiently within real deployment constraints. We introduce Kairos, a native world model stack designed around these requirements. (1) Kairos learns the world by pioneering a Native Pre-training Paradigm governed by a Cross-Embodiment Data Curriculum, which organizes open-world videos, human behavioral data, and robot interactions into a progressive developmental pathway. (2) Kairos maintains the world by unified world understanding, generation, and prediction within a Native Unified Architecture equipped with Hybrid Linear Temporal Attention, where sliding-window attention captures local dynamics, dilated sliding windows capture mid-range dependencies, and gated linear attention maintains persistent global memory. We establish formal theoretical bounds demonstrating that this temporal factorization strictly limits error accumulation, mathematically guaranteeing state propagation across extended horizons. (3) Kairos runs the world by incorporating a Deployment-Aware System Co-Design to support low-latency rollout generation on server and consumer-grade hardware for real-world observation-action-feedback loops. Experiments on embodied world-model, long-horizon, and action-policy benchmarks show that Kairos achieves top level performance while offering a strong efficiency-capability trade-off. Together, these results position Kairos as a cohesive operational foundation for future self-evolving physical intelligence.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 2.1 核心架构：原生预训练 + 统一架构 + 部署感知协同设计

Kairos包含三个紧密耦合的支柱：

### 支柱一：原生预训练范式（Cross-Embodiment Data Curriculum, CEDC）

- **核心理念**：拒绝后训练/微调范式，而是从零开始通过渐进式数据课程注入物理和具身知识。
- **三个阶段**：
    - **阶段I：物理预训练**——使用百万小时级开放世界视频，内化重力、碰撞等物理规律；采用渐进式分辨率策略（256p→720p）、基于流匹配（Flow Matching）的训练目标，并引入形状感知时间步长偏移（shape-aware exponential timestep shifting）以适配不同视频长度和分辨率。
    - **阶段II：人类中心行为预训练**——使用超10万小时人类演示数据（第一/第三人称），学习任务结构和行为因果；通过指令式字幕逐步增强指令跟随能力。
    - **阶段III：联合世界-动作训练**——同时优化Video DiT（视觉生成）和Action DiT（动作预测），采用混合注意力掩码机制，使动作预测能利用全部历史观察，但不依赖于未来视频生成，实现联合优化且推理时可独立执行动作预测。

### 支柱二：原生统一架构（Hybrid Linear Temporal Attention）

- **统一理解-生成-预测**：使用VLM（Qwen系列）作为世界理解模块；Diffusion Transformer（DiT）作为生成主干；Mixture-of-Transformers（MoT）实现世界预测（WAM）。
- **混合线性时间注意力**（LinearDiT）：
    - **滑动窗口注意力（SWA）**：捕获局部动态（短程）。
    - **扩张滑动窗口注意力（DSWA）**：捕获中程交互（秒量级）。
    - **门控线性注意力（GLA，基于GatedDeltaNet）**：作为全局因果记忆，具有线性复杂度，并通过Δ更新规则和遗忘门实现可控长期记忆。
- **理论证明**：论文在附录B中严格证明：
    - **定理1（必要性）**：若最优预测器依赖于超出近期窗口的历史，则任何窗口受限预测器必然产生不可消除的超额风险（严格正误差），证明持久潜状态是必要的。
    - **定理2（充分性）**：若最优预测器可分解为共享状态、短程、中程和全局门控记忆，且全局记忆更新是收缩的（ρ<1），则混合多尺度记忆的长期超额风险有界，误差由局部分支近似质量和全局记忆扰动项控制。

### 支柱三：部署感知系统协同设计

- **时间步蒸馏**：使用分布匹配蒸馏（DMD）和一致性蒸馏（CM）将教师模型压缩为4步生成器，保持视觉质量。
- **硬件感知优化**：
    - 服务器端：混合并行（序列并行+张量并行）、DiT-Cache、算子融合。
    - 消费级设备：FP8低精度计算（INT4/INT8量化QK，FP8做PV矩阵乘）、权重量化文本编码器、分块流式访问。
- **效率对比**：在720P 5秒生成任务中，Kairos-4B单GPU耗时43秒（较Cosmos 2.5-14B快28-85倍，较Wan2.2-5B快2.5-3.7倍），且延迟随视频时长线性增长。

### 训练目标

- 采用流匹配（Flow Matching）的MSE损失，统一用于图像/视频生成。第三阶段联合损失为 L = L_video + λ L_action。

- **总体实验数量**：非常充分，覆盖了5个主要基准的多个子集（如WorldModelBench的机器人子集和完整集、PAI-Bench的5秒和15秒版本、VideoPhy的344个测试prompt），并包含大量对比（10+个基线）和消融研究（人类数据缩放、VLM升级、联合训练、联合去噪）。
- **公平性**：
    - 所有模型在相同评估协议下进行，部分结果由团队复现（标注*）。
    - 在粗粒度指标（如平均分）领先的前提下，还进行了人类评估（10名志愿者，匿名排序），进一步验证了主观质量。
    - 消融实验设计合理，隔离各因素。
- **客观性**：自动化评估采用标准工具（VBench、VILA评估器等），人类评估为双盲评分，总体客观可信。

## 6. 论文的主要结论与发现

1. **SOTA性能**：Kairos-4B在WorldModelBench机器人子集（总分9.30）、DreamGen Bench（平均分0.618）、PAI-Bench机器人子集（总分82.57）、LIBERO-Plus（90.8平均率）、RoboTwin 2.0（96.1平均率）上均达到或超过现有模型，尤其在指令跟随和物理一致性上表现突出。
2. **高效率-能力权衡**：相比大参数模型（如14B+），Kairos-4B以4B参数实现相近或更优性能，且推理速度呈线性扩展（延迟随视频时长线性增长），远优于同类对角时间注意力模型（二次复杂度）。
3. **理论验证**：严格证明了持久潜状态对于长时程一致性的必要性，以及混合多尺度记忆的近似充分性，为架构设计提供了数学基础。
4. **跨具身泛化能力**：在多个机器人形态（单臂、双臂、灵巧手、人形）上展示了高质量视频生成和动作执行，验证了数据课程和原生预训练的有效性。
5. **自进化框架验证**：通过prompt自对齐实验初步验证了自我进化循环的可行性（理解模块作为内部分数函数进行迭代优化）。

## 7. 优点：方法或实验设计上的亮点

- **系统级设计**：同时解决学习、维护、运行三个层面，而非单一模块优化。
- **理论驱动架构**：提供完整的必要性-充分性理论证明，将局部窗口不足和混合注意力优势数学化，超越经验调优。
- **部署优先原则**：从训练到推理均考虑边缘部署，包括蒸馏、量化、流式访问、低精度计算，实际测试在消费级GPU上运行。
- **数据工程创新**：提出Diverse-and-Conquer字幕生成、物理中心化和长时程CoT文本增强，提升物理理解和任务规划能力。
- **实验全面且深入**：覆盖多模态（视频生成、机器人操控、物理常识推理）、多尺度（5秒/15秒）、多评价维度（自动化+人类评估），消融丰富。
- **代码与模型开源**：提供GitHub、HuggingFace、ModelScope链接，促进可重复性。

## 8. 不足与局限

- **未公布训练算力细节**：论文未明确给出GPU型号、数量、训练时长等，难以复现训练成本。
- **自进化能力尚未全面验证**：论文提出自我进化框架，但仅展示了prompt自对齐的初步实验，未实现完整的闭环自我进化（如通过真实环境交互持续改进策略），这是未来工作。
- **高难度物理场景覆盖有限**：虽然在VideoPhy上达到最高，但物理常识平均分仍低于50%，说明复杂物理交互（如柔性体、流体、耦合系统）的建模仍有提升空间。
- **基准多样性局限**：主要聚焦在机器人操作和简单物理场景，未涉及自动驾驶、建筑环境、医疗等更广泛物理AI领域（PAI-Bench包含自动驾驶等，但机器人子集为重点）。
- **模型规模限制**：当前模型为4B参数，虽然效率高，但在极端高分辨率、超长视频（分钟级）生成上的能力未测试，且可能受限于内存和延迟。
- **消融实验未覆盖所有组件**：例如未单独消融SWA/DSWA/GLA的各自贡献，而是整体对比混合注意力 vs 全注意力；也未消融蒸馏策略中各组件（DMD vs CM）的独立影响。
- **潜在数据偏差**：自采数据来源和过滤规则可能存在地域/内容偏好，且人类数据主要来自第一/第三人称，对某些文化背景下的行为泛化性未知。
