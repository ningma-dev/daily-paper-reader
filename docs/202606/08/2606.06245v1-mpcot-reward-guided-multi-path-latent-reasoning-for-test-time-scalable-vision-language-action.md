## 摘要
视觉-语言-动作（VLA）策略在长期和高不确定性的控制中仍然脆弱，因为单次动作解码提供的推断时间推演有限。显式的思维链可以增加推理深度，但会引入令牌延迟和间接的文本到动作接口。我们提出了MPCoT，一种奖励引导的多路径潜在推理框架，该框架初始化M个假设，通过K个权重共享步骤对其进行优化，并在动作解码前进行软聚合。一个仅用于训练的路径偏好目标通过专家动作一致性、基于世界模型/VLM的进度和成功反馈来评估候选动作分支，以将潜在路径评分器与下游执行质量对齐。MPCoT保留了原始的8步动作接口，生成零推理令牌，并提供了可配置的推理控制（K，M）。在LIBERO和CALVIN上的匹配协议下，MPCoT提高了长期性能，消融实验证实了深度-宽度效应、置信度加权聚合和奖励引导的路径监督。

## Abstract
Vision-Language-Action (VLA) policies remain brittle in long-horizon and high-uncertainty control, where one-pass action decoding provides limited inference-time deliberation. Explicit chain-of-thought can increase reasoning depth, but introduces token latency and an indirect text-to-action interface. We propose MPCoT, a reward-guided multi-path latent reasoning framework that initializes $M$ hypotheses, refines them for K weight-tied steps, and softly aggregates them before action decoding. A training-only path-preference objective evaluates candidate action branches with expert-action consistency, world-model/VLM-based progress, and success feedback to align the latent path scorer with downstream execution quality. MPCoT preserves the original 8-step action interface, generates zero reasoning tokens, and exposes configurable inference controls (K,M). Under matched protocols on LIBERO and CALVIN, MPCoT improves long-horizon performance, with ablations confirming depth-width effects, confidence-weighted aggregation, and reward-guided path supervision.

# 论文总结：MPCoT: Reward-Guided Multi-Path Latent Reasoning for Test-Time Scalable Vision-Language-Action

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有的视觉-语言-动作（VLA）策略在长程（long-horizon）和高不确定性的控制任务中表现脆弱，其单次动作解码（one-pass action decoder）提供的推理深度有限。早期小误差容易在后续动作序列中累积放大。
- **现有方法局限**：显式思维链（explicit CoT）可以增加推理深度，但会引入中间文本/令牌延迟和间接的文本-动作接口，增加推理开销。纯隐式策略效率高但缺乏多步推理能力。
- **论文出发点**：探索在连续潜在空间中进行多路径推理的中间方案，即在不改变原始动作接口、不生成推理令牌的前提下，通过可配置的深度（K步细化）和宽度（M条假设）扩展测试时计算量，提升VLA在长程任务上的鲁棒性。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：MPCoT 在策略解码前插入一个轻量级潜在推理模块，该模块初始化 M 条不同的潜在假设路径，用权重共享的细化算子进行 K 步迭代修正，然后通过置信度加权（soft aggregation）合并这些路径，最后输入给不变的动作头解码动作。训练时额外引入奖励引导的路径偏好学习，以使路径评分器与下游执行质量对齐。
- **关键技术细节**：
  - **潜在假设初始化**：基于多模态融合后的上下文向量 $c_t$，加上可学习的假设码 $e_m$ 和训练时小扰动 $\epsilon_{t,m}$ 生成 M 条初始潜在状态 $z_{t,m}^{(0)} = c_t + e_m + \epsilon_{t,m}$。
  - **权重共享的迭代细化**：对每条路径，使用同一个轻量残差 MLP $F_\theta$ 进行 K 步更新：$r_{t,m}^{(k)} = F_\theta([\text{LN}(z_{t,m}^{(k-1)});c_t])$，$z_{t,m}^{(k)} = z_{t,m}^{(k-1)} + r_{t,m}^{(k)}$。每步均以 $c_t$ 为条件防止推理漂移。
  - **软聚合**：使用两层评分器 $h_\omega$ 为每条最终路径 $z_{t,m}^{(K)}$ 计算 softmax 权重 $p_\omega(m)$，加权求和得到聚合潜在表示 $\bar{z}_t^{(K)} = \sum_m p_\omega(m) z_{t,m}^{(K)}$。
  - **奖励引导路径偏好学习（仅训练阶段）**：训练时将每条候选路径解码为动作块，由三个奖励信号评估：
    - 专家动作一致性奖励 $r_t^{\text{act}}$（负 L2 距离）；
    - 世界模型/VLM 进度奖励 $r_t^{\text{prog}}$（基于预测的后续场景进度变化）；
    - 成功奖励 $r_t^{\text{succ}}$（稀疏标签）。
    将奖励转换为优势 $A_{t,m}$（减去路径平均基线），训练路径评分器以最大化被采样的正确路径的对数概率（stop-gradient防止梯度回传至奖励计算）。
  - **总训练损失**：$\mathcal{L} = \mathcal{L}_{\text{BC}} + \lambda_{\text{RL}}\mathcal{L}_{\text{RL}} + \lambda_{\text{div}}\mathcal{L}_{\text{div}}$，其中 $\mathcal{L}_{\text{div}}$ 为路径多样性正则（指数余弦相似度惩罚）。
  - **推理时**：移除所有奖励、世界模型评估器和回报计算，仅使用已学习的评分器进行软聚合，Zero 推理令牌，保持原 8 步动作块输出接口。

## 6. 主要结论与发现
- **长程任务显著提升**：在 CALVIN 5 步成功率上，MPCoT 达 89.4%，比 OpenVLA-OFT（72.9%）高出 16.5 个点，比最强基线 AVA-VLA（84.1%）高出 5.3 个点。
- **深度与宽度互补**：当 M=1 时，K 从 1 增至 5 持续提升；固定 K=3 时，M 从 1 增至 4 进一步提升。最佳组合 $(K,M)=(5,4)$ 在 LIBERO 平均 SR 达 98.9%，Long SR 达 98.9%。
- **软聚合优于硬选择**：软聚合加多样性正则（表 A.3）使路径相似度从 0.91 降至 0.66，稳定性提升（std 从 0.41 降至 0.18）。
- **奖励引导路径偏好有效**：全奖励（动作一致性+进度+成功）使路径一致性（Path Consistency）从 68.5%（无奖励）提升至 84.3%，并带来最佳长程性能（见表 5）。
- **零推理令牌、低延迟**：相比显式 CoT 基线（110-160ms），MPCoT 仅增加 14ms（24ms→38ms），且无令牌生成。
- **可配置测试时扩展**：通过调整 K 和 M，可在推理时灵活分配计算量，实现性能-效率折中。

## 7. 优点
- **接口无损**：保持原始 VLA 策略的 8 步动作块输出，零推理令牌，不需要改变策略架构或推理流程。
- **轻量高效**：仅增加约 2.7% 参数（权重共享 MLP + 两层评分器），推理开销线性于 $K\times M$。
- **奖励引导的隐式对齐**：训练时仅学习路径偏好，推理时完全无奖励/世界模型调用，保证推理纯净。
- **可解释的控制变量**：深度（K）和宽度（M）提供了明确的测试时计算-性能折中，便于部署调整。
- **充分消融**：系统消融了每个组件（深度、宽度、聚合方式、奖励信号类型），证实了设计的必要性。
- **在多个基准上达到 SOTA**：在 LIBERO 和 CALVIN 上均显著超越近期最强方法（如 AVA-VLA、UnifiedVLA 等）。
