## 摘要
世界模型使智能体能够基于动作预测未来动态，这使得隐变量表示的选择成为规划与控制的核心。这类表示通常要么从语义结构有限的像素直接学习，要么继承自包含过多与任务无关细节的冻结视觉基础模型，导致状态空间与下游规划和控制匹配不佳。这在无奖励离线设置中尤为挑战，因为模型必须从固定轨迹中学习，而没有奖励监督或在线交互。为解决此问题，我们提出TC-WM，一个将基础模型嵌入转化为紧凑、任务充分的世界表示的框架。关键设计是将预训练嵌入空间视为语义脚手架而非最终状态空间：TC-WM 将高维视觉嵌入线性投影为紧凑隐变量作为动态空间，通过对比学习使子空间与智能体物理状态对齐，并重构嵌入以保留有用的视觉结构。这结合了基础特征的通用性与以任务为中心的动态的可控性。理论上，我们证明TC-WM足以识别底层以任务为中心的隐变量因子（仅需简单变换）。实验上，TC-WM在多种环境（如Robomimic和D4RL）中实现测试时规划，相比最先进方法获得了更好的世界建模质量和更精确的控制。

## Abstract
World models enable agents to predict future dynamics conditioned on actions, making the choice of latent representation central to planning and control. Such representations are often either learned directly from pixels with limited semantic structure or inherited from frozen visual foundation models with excessive task-irrelevant detail, yielding state spaces that are poorly matched to downstream planning and control. This is especially challenging in reward-free offline settings, where the model must learn from fixed trajectories without reward supervision or online interaction. To address this, we propose TC-WM, a framework for turning foundation-model embeddings into compact, task-sufficient world representations. The key design is to treat the pretrained embedding space as a semantic scaffold rather than as the final state space: TC-WM linearly projects high-dimensional visual embeddings into a compact latent as the dynamic space, aligns a subspace with the agent's physical state via contrastive learning, and reconstructs embeddings to preserve useful visual structure. This combines the generality of foundation features with the controllability of task-centric dynamics. Theoretically, we show that TC-WM suffices to identify the underlying task-centric latent factors up to a simple transformation. Empirically, TC-WM enables test-time planning across diverse environments (e.g., Robomimic and D4RL), achieving better world-modeling quality and more precise control than state-of-the-art approaches.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：世界模型的潜在状态表示是关键，但现有方法存在两难：要么直接从像素学习（语义结构弱），要么继承自冻结的视觉基础模型（如DINO）——虽然语义丰富，但包含大量与任务无关的细节（纹理、光照、背景等），导致状态空间与下游规划控制不匹配。
- **挑战加剧场景**：在**无奖励离线设置**中，模型只能从固定轨迹学习，没有奖励信号或在线交互来引导表示学成与任务相关，因此对潜在表示的任务充分性要求更高。
- **核心动机**：视觉基础模型嵌入是良好的语义坐标系，但不应直接作为动态空间；需要将其压缩为**任务充分的紧凑隐变量**，同时保留规划和控制所需的结构信息。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- **TC-WM（Task-Centric World Model）**：将预训练视觉嵌入视为可复用的语义脚手架（而非最终状态空间），在其上学习紧凑的隐变量动态空间。
- **关键设计**：
  - 线性投影高维嵌入 → 紧凑隐变量 `z_t`
  - 将`z_t`分解为任务相关子空间`z^s_t`和补充子空间`z^c_t`
  - 通过对比学习（InfoNCE）将`z^s_t`与物理本体感信号`sp_t`对齐
  - 通过解码器重构视觉嵌入，保留有用视觉结构

### 2.2 关键技术细节

#### ① 联合嵌入（Joint Embedding）
- 冻结视觉编码器（默认DINOv2）提取图像嵌入`f^o_emb(o_t)`
- 可训练线性层`f^p_emb(sp_t)`编码本体感
- 拼接得到`x_t = [f^o_emb(o_t), f^p_emb(sp_t)]`

#### ② 隐变量编码器（Latent Encoder）
- `z_t ~ q(z_t | x_t)`，线性投影到紧凑空间（维度远小于嵌入）

#### ③ 任务中心对齐（TC Alignment）
- 分解`z_t = [z^s_t, z^c_t]`
- 使用InfoNCE对比损失对齐`z^s_t`与`sp_t`：
  ```
  L_align = -log exp(sim(g_φ(z^s_t), h_ψ(sp_t))/τ) / Σ exp(sim(...))
  ```
- 通过`ℓ1`惩罚自动选择任务相关维度，实现稀疏子空间选择

#### ④ 隐变量动态（Latent Dynamics）
- Transformer架构`f_θ`预测下一时刻隐变量：
  ```
  L^z_dyn = || ẑ_{t+1} - q(z_{t+1}|x_{t+1}) ||²₂
  ```

#### ⑤ 任务中心动态（TC Dynamics）
- 直接预测原始本体感（而非嵌入），防止退化：
  ```
  L^s_dyn = || f^p_η(z_{t-H:t}, a_{t-H:t}) - sp_{t+1} ||²₂
  ```

#### ⑥ 嵌入解码器（Embedding Decoder）
- 线性解码器`f_dec`重构视觉嵌入：
  ```
  L_rec = || ẑx_t - x_t ||²₂
  ```
- 作用：作为正则项，防止紧凑隐变量坍塌

#### 总体损失：
```
L_all = L^z_dyn + L^s_dyn + λ_align L_align + λ_rec L_rec
```

### 2.3 规划与控制的两种方式

| 任务类型 | 规划方法 | 说明 |
|---------|---------|------|
| 低维动作空间（导航、DMC连续控制） | CEM（交叉熵方法） | 基于模型预测控制，滚动K条动作序列，以目标隐变量距离为代价 |
| 高维动作空间（操控） | LDP（隐变量扩散规划） | 训练扩散规划器生成未来隐变量轨迹，逆动态模型恢复动作 |

### 2.4 理论保证
- **定理1**：在一定假设下，TC-WM学习的紧凑隐变量`z_t`是块状可识别的，任务中心子空间`z^s_t`是仿射可识别的（可恢复至线性变换）。
- 证明思路：利用条件独立性和积分算子对角化，InfoNCE对齐将被动假设转化为主动构建的多样性。

## 6. 主要结论与发现

1. **世界模型预测**：TC-WM在隐变量预测误差上显著低于所有基线（Fig.6），尤其是在操控任务中；图像重构竞争力稍弱但可接受（紧凑投影舍弃低层细节）。
2. **规划性能**：
   - 在高维动作空间操控任务（LDP规划）中，TC-WM超越所有基线，且是唯一所有任务均优于DINO-WM的方法（Fig.8）。
   - 在低维导航/控制任务（CEM）中，TC-WM与最强基线持平。
3. **表示质量**：
   - 线性探测R²显示TC-WM能更准确恢复物理状态（Fig.2）。
   - 有效秩显示TC-WM对抗坍塌能力更强（Fig.2）。
4. **理论**：定理1为紧凑隐变量可识别性提供了理论支撑，解释了线性投影+对齐为何能恢复任务相关因子。
5. **消融**：嵌入重构损失`L_rec`是防止坍塌的关键；本体感对齐对规划性能至关重要；线性投影比MLP/VAE/ViT更优。

## 8. 不足与局限

- **依赖本体感信号**：TC-WM需要训练时提供物理状态信号（如关节位置、末端执行器位姿）。作者指出若不可用，可用深度估计、多视角一致性等替代，但未验证。
- **模拟环境局限**：所有实验在模拟器中进行，真实机器人部署存在域差，未见量化评估。
- **数据来源偏差**：
  - DMC数据集由TD-MPC2策略生成，可能偏好高质量轨迹，影响泛化。
  - Robomimic仅200条专业演示，规模较小，复杂场景泛化性存疑。
- **图像重构质量**：在视觉复杂场景（如Push-T、Cheetah）中，图像重构误差高于某些基线（DreamerV3等），因为紧凑投影丢弃了低层细节——这是设计取舍，但可能影响像素级任务。
- **未对比最新多模态世界模型**：如V-JEPA 2（发表于2025年6月）未被包含在对比中，其实验时间线可能存在滞后。
- **计算资源**：虽然单卡可训练，但论文未在小资源场景下评估（如低端GPU），且未讨论数据效率（需要多少轨迹才能达到当前性能）。
